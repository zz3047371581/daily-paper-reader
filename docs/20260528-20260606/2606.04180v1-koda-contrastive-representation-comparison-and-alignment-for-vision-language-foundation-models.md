## 摘要
视觉-语言基础模型，如CLIP和SigLIP，为多模态学习系统提供了广泛使用的表示。尽管这些模型通常通过下游性能进行比较，但此类评估往往无法解释它们的表示在结构上有何不同。在本工作中，我们通过对比嵌入聚类任务来研究该问题：识别在一种表示下弱聚类而在另一种表示下强聚类的样本子集。我们提出了“差异分析的核优化（KODA）”，一个基于核的对比表示比较与对齐框架。KODA通过模态核组合构建统一的多模态核，并将差异发现形式化为一个约束优化问题，该问题在参考表示中抑制相干性的同时，搜索一个表示中的相干结构。这产生了与特定样本子集和模态交互相关联的可解释差异方向。为了将KODA扩展到大规模视觉-语言数据集，我们利用随机投影（包括用于平移不变核的随机傅里叶特征）开发了联合核的随机低维近似。经验表明，KODA识别出了跨视觉-语言表示的一致且可解释的差异结构，并为表示对齐提供了样本子集。代码可在https://github.com/yokiwuuu/KODA获取。

## Abstract
Vision-language foundation models such as CLIP and SigLIP provide widely used representations for multimodal learning systems. While these models are typically compared through downstream performance, such evaluations often do not explain how their representations differ structurally. In this work, we study this problem through the task of Contrastive Embedding Clustering: identifying sample subsets that are weakly clustered under one representation but strongly clustered under another. We propose \emph{Kernel Optimization for Discrepancy Analysis (KODA)}, a kernel-based framework for contrastive representation comparison and alignment. KODA constructs unified multimodal kernels through modality-wise kernel composition and formulates discrepancy discovery as a constrained optimization problem that searches for coherent structures in one representation while suppressing coherence in a reference representation. This yields interpretable discrepancy directions associated with specific sample subsets and modality interactions. To scale KODA to large vision-language datasets, we develop randomized low-dimensional approximations of joint kernels using random projections, including Random Fourier Features for shift-invariant kernels. Empirically, KODA identifies consistent and interpretable discrepancy structures across vision-language representations and provides sample subsets for representation alignment. The code is available at https://github.com/yokiwuuu/KODA.

本文首次提出 **对比嵌入聚类（Contrastive Embedding Clustering）** 任务：识别在一个表示下呈现强聚类、而在另一个表示下被抑制或散乱的样本子集。围绕该任务，提出框架 **KODA（Kernel Optimization for Discrepancy Analysis）**，旨在通过核优化的方式从几何上自动发现两个表示之间的结构性差异，并利用这些差异解释模型行为、辅助表示对齐。

## 2. 方法论

### 2.1 核心思想与公式

给定两个表示 \(A,B\) 在同一参考数据集 \(\{x_i\}_{i=1}^n\) 上诱导的归一化核矩阵 \(K_A, K_B\)，KODA 将对比嵌入聚类形式化为带约束的二次优化问题：

\[
\max_{x\in\mathbb{R}^n} x^\top K_A x \quad \text{s.t.} \quad x^\top K_B x \le \epsilon,\; \|x\|_2=1.
\]

- 目标项 \(x^\top K_A x\) 衡量方向 \(x\) 在表示 \(A\) 下的聚类强度。
- 约束项 \(x^\top K_B x \le \epsilon\) 强制该方向在参考表示 \(B\) 下高度散乱（弱聚类）。
- 优化问题非凸，但可通过 KKT 条件转化为广义特征值问题 \((K_A - \lambda K_B)x = \nu x\)，其中 \(\lambda\ge0\) 由一维搜索确定。

### 2.2 迭代多方向提取

为发现多个差异模式，KODA 迭代求解上述问题，并施加正交性约束以屏蔽已找到的方向：

\[
P_{t-1}=I-U_{t-1}U_{t-1}^\top,\quad
\max_{x} x^\top (P_{t-1}K_A P_{t-1})x \quad \text{s.t.} \quad x^\top (P_{t-1}K_B P_{t-1})x \le \epsilon,\; \|x\|_2=1.
\]

### 2.3 多模态扩展：乘积核与协方差块加速

对于图像‑文本配对数据 \((u_i,v_i)\)，KODA 通过模态核乘积构建联合核：

\[
k((u,v),(u',v')) = k_x(u,u') \cdot k_t(v,v').
\]

对应的核矩阵为 \(K = K_x \odot K_t\)。为避免张量积特征映射的维度爆炸，采用 **联合随机傅里叶特征** 近似核矩阵：

\[
\varphi(u,v) = \frac{1}{\sqrt{r}}\big[\cos(\omega_{x,1}^\top u + \omega_{t,1}^\top v), \sin(\dots),\dots\big] \in \mathbb{R}^{2r}.
\]

令 \(\Phi\) 为该特征矩阵 (\(n\times 2r\))，核矩阵近似为 \(\widetilde{K} = \Phi\Phi^\top\)。进而，将主导特征向量计算从 \(n\times n\) 降至协方差块 \((d_1+d_2)\times(d_1+d_2)\)，使 KODA 在万级以上样本量下仍可运行。

理论保证方面给出了：
- 协方差块主导特征向量的样本复杂度（接近 \(O(\sqrt{d/n})\)）；
- 联合随机傅里叶近似在 Frobenius 范数下的集中速率 \(O(1/\sqrt{r})\)，以及引出的特征空间稳定性。

## 6. 主要结论与发现

- KODA 能够**无监督、可解释地**定位两个表示之间高度不对称的样本子集。单模态下，DINOv2 与 CLIP 的差异方向对应明确的语义类别（如狗品种），多模态下，不同视觉‑语言模型在体育活动、动物、室内场景等概念上表现出组织方式的分化。
- 与基于核差谱分解的 SPEC 相比，KODA 发现的差异方向在 Rayleigh 商指标上具有更显著的强弱对比，表明约束优化有效地抑制了参考表示中的残余结构，提取出真正与该表示“弱相关”的模式。
- KODA 筛选的样本子集能够明显**放大模型之间的检索性能差距**，并可作为有效切片用于**定向表示对齐**：在 KODA 选出的样本上微调弱模型，可大幅提高其与目标模型在聚类结构上的吻合度（例如 AMI 从 0.25 提升至 0.78）。
- 理论分析支持协方差块降维和随机傅里叶近似在大样本下的正确性，实践上能稳定地处理数万级样本。

- **任务定义新颖**：首次提出对比嵌入聚类，明确区分“强聚类”与“弱聚类”，超越全局距离度量的比较。

- **实用性强**：不仅能诊断差异，还能驱动目标对齐，为模型选择、数据策划、表示融合提供了新的工具链。

## 8. 不足与局限

- **依赖共享参考集**：KODA 要求两个表示在同一组样本上计算核矩阵，不能直接处理输入空间不同或缺乏配对的情况。
- **无监督差异可能不完全对应任务相关差异**：当前发现的差异方向未必与特定下游任务的性能差异直接挂钩，缺乏任务条件或监督信号的引入。
- **计算资源描述不充分**：尽管使用了随机特征降维，但未给出大规模实验的具体时间成本、内存占用或可扩展性上限，使读者难以评估实际应用门槛。
- **对核函数和超参数敏感**：消融实验表明核函数（高斯 vs 余弦）会显著影响发现的差异方向；带宽选择依赖经验策略，可能缺乏自适应性。
- **统计解释需加强**：虽对样本子集聚类质量进行了评估，但对所发现方向的“显著性”或置信度缺乏严格的统计检验。
- **仅评估了有限的模型组合**：多模态实验集中在 CLIP、BLIP、SigLIP 等几类模型，尚未拓展到更广泛的生成式模型或大规模基础模型。
- **对齐实验有限**：对齐仅显示了微调后的聚类一致性提升，未对比其他对齐方法，也未分析对下游任务准确性的影响。
