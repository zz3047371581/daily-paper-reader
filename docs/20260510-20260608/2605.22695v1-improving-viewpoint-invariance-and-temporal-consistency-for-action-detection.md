## 摘要
视角变化不变性和动作时序一致性对于在未裁剪视频中有效部署人类动作检测至关重要。现有的基于外观的视频检测方法在训练过程中常因视角多样性有限而受限，而基于运动的检测方法往往无法建模跨连续运动窗口的细粒度时序关系。本文提出了一种新颖的两阶段动作检测方法，旨在同时提升视角不变性和全局时序一致性。在第一阶段，我们从增强的虚拟视角中提取运动特征，这些特征仅在训练时使用。随后，第二阶段引入了一种基于选择性状态空间序列建模的新型视角不变多尺度时序编码器，以聚合跨视角和时间尺度的信息。在PKU-MMD和BABEL基准上的实验表明，该方法在所有考虑的评估方案下均显著优于当前最先进的方法。代码和训练模型可在以下网址获取：https://icb-vision-ai.github.io/HydraView-TAD

## Abstract
Viewpoint change invariance and action temporal consistency are critical aspects for the effective deployment of human action detection of untrimmed videos. Existing appearance-based video detection methods often struggle with limited viewpoint diversity during training, while motion-based detection approaches frequently fail to model fine-grained temporal relationships across consecutive motion windows. This paper introduces a novel two-stage action detection approach designed to improve both view-invariance and global temporal coherence properties. In the first stage, we extract motion features from augmented virtual viewpoints, solely used at training. Then, the second stage introduces a new view-invariant, multi-scale temporal encoder based on selective state-space sequence modelling to aggregate information across viewpoints and time scales. Experiments on PKU-MMD and BABEL benchmarks demonstrate that this approach significantly outperforms state-of-the-art methods in all considered splits. Code and trained models are available at: https://icb-vision-ai.github.io/HydraView-TAD

## 1. 核心问题与整体含义

- **核心问题**：未裁剪视频中的人类动作检测（Temporal Action Detection, TAD）面临两大关键挑战：
  - **视角变化**：摄像机视角的改变会导致动作外观发生显著变化，破坏动作表示的一致性，降低检测精度。
  - **时序一致性**：动作的演变在相邻帧间通常具有平滑的时序特性，但现有基于运动的方法往往忽视跨短时窗口的细粒度时序关系，导致边界定位不准。
- **研究动机**：现有基于外观的视频检测方法在训练时视角多样性有限，泛化性不足；基于运动（骨架）的方法虽对视角有一定鲁棒性，但未能有效建模长程时序依赖。因此，需要设计一个能同时强化视角不变性和全局时序一致性的统一框架。
- **整体含义**：本文提出一种两阶段方法，仅在训练阶段利用多虚拟视角增强运动特征学习，并引入基于选择性状态空间模型的多尺度时序编码器，在推理阶段仅使用单视角视频即可实现高精度、视角鲁棒的动作检测。

## 2. 方法论

### 核心思想
- 将输入视频分割为短时窗口（16帧），分两阶段处理：
  1. **窗口运动特征编码**：在训练时通过模拟多虚拟摄像机生成不同视角的2D骨架投影，迫使空间编码器学习视角不变的运动表示，并提升窗口内的时序一致性。
  2. **多视角多尺度时序编码**：利用线性复杂度的状态空间序列模型（Mamba）同时沿着视角维度和时间维度进行双向扫描，聚合跨视角和多时间尺度的信息，最终输出逐帧多标签动作预测。

### 关键技术细节

#### 第一阶段：窗口运动特征编码（SWGCN）
- **输入**：3D骨架序列（F帧，J个关节）。每个窗口 `W ∈ R^{F×J×3}`。
- **视角增强**：通过V个虚拟摄像机投影生成V个虚拟2D骨架序列，模拟自遮挡（利用躯干平面判断遮挡）。
- **编码器**：改造的时空图卷积网络（Spatial-Window GCN, SWGCN），针对短窗口做出如下调整：
  - 去除时间下采样以保持帧级分辨率（步长=1）。
  - 调整时间卷积核尺寸以匹配窗口长度（避免感受野超出窗口导致过平滑）。
  - 使用组归一化（GroupNorm）替代批归一化，以应对小批量训练带来的统计不稳定。
- **训练策略**：先用交叉熵损失训练SWGCN，冻结权重后提取窗口级特征向量`M_v`。

#### 第二阶段：多视角多尺度时序编码器（HydraView）
- **输入**：多视角窗口特征 `M ∈ R^{V×T×C}`（V=视角数，T=时间窗口数，C=特征维度）。
- **HydraView结构**：由多个ViewMamba块组成，每个块负责一个特定的时空尺度。
  - **ViewMamba块**：
    1. **2D卷积投影**：沿视角维度应用步长 `s_v` 的卷积，压缩视角数量但保留时间分辨率。公式：
       `M^{(s_v)}_{k,t} = Σ_{i,j} W_{i,j} M_{k s_v + i, t + j}`
    2. **时间扩张**：以扩张因子 `s_t` 采样时间维度，扩大有效时序感受野。
    3. **多视角扫描SSM**：采用选择性状态空间模型（Mamba），进行四方向扫描：
       - 沿视角维度：前向和后向。
       - 沿时间维度：左到右和右到左。
       这一设计允许模型同时捕获视角间的互补信息和时序中的前后依赖，对遮挡或缺失检测具有鲁棒性。
       - SSM动态：`h_t = A h_{t-1} + B u_t`，`y_t = C h_t + D u_t`，其中A, B, C, D为输入依赖参数。
  - **多尺度设置**：使用3个ViewMamba块，第s个块以步长s处理序列，学习不同时间尺度的关系。
- **多尺度融合器**：各尺度输出经Vision Mamba层（沿尺度维度双向扫描）融合后，由MLP头输出多标签分类结果。
- **训练损失**：每个时间步为所有动作类应用独立的二元交叉熵损失（BCE）。

### 公式或算法流程简述（文字描述）
1. 将骨架序列切分为固定窗口。
2. 训练阶段：每个窗口通过虚拟摄像机投影得到V个变体 → SWGCN提取窗口特征 → 冻结SWGCN。
3. 多窗口特征组成多视角时序张量 → HydraView在多个尺度上通过ViewMamba块处理：
   - 2D卷积降视角数 → 时间扩张 → SSM四向扫描。
4. 各尺度特征融合 → 逐帧预测动作类别。

## 8. 不足与局限

- **对3D骨架输入的依赖**：方法要求输入3D骨骼数据，这需要额外的姿态估计步骤或特定采集设备，限制了直接应用于普通RGB视频的场景。
- **虚拟视角模拟的保真度**：模拟的2D投影和遮挡模型可能无法完全代表真实摄像机中的复杂的透视、分辨率变化和噪声。

- **实验规模有限**：训练仅在单块RTX 5000 Ada GPU上进行，未报告更大批次或分布式训练的影响；未在更大规模的数据集（如Kinetics-TAD）上验证。
- **边界模糊问题**：定性分析中仍存在边界偏移和类别混淆（如“turn”和“stand”性能相对较低），长序列中的连续误检仍有改进空间。
