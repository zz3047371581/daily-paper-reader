## 摘要
最近视频大语言模型（Video-LLMs）的进展极大地扩展了多模态推理能力。然而，从长视频序列中提取的大量视觉令牌带来了难以承受的计算成本，限制了它们在实际场景中的部署。现有的无需训练的令牌压缩方法根据注意力大小作为语义重要性的代理来选择令牌，但往往忽视位置偏置，仅依赖短期的时间局部性，导致冗余的时空覆盖和低效的令牌使用。我们提出了DynaTok，一个无需训练、时间自适应且偏置感知的令牌压缩框架，它在时间和空间两个维度上分配令牌预算。通过一个轻量级的指数移动平均（EMA）记忆模块，时间预算分配（TBA）模块动态地为冗余帧分配更少的令牌，为新颖帧分配更多的令牌，捕捉长期的时间变化。空间预算分配（SBA）模块通过使用基于激活的注意力图选择空间多样且语义重要的特征，同时利用空间记忆来减少先前选择区域的冗余并缓解位置偏置，从而补充时间维度。DynaTok可以无缝集成到现有的Video-LLMs中，如LLaVA-OneVision和LLaVA-Video，无需重新训练，并且在激进压缩下有效保持语义覆盖。在四个代表性的视频问答基准——MVBench、LongVideoBench、MLVU和VideoMME上的实验表明，即使减少了90%的令牌，DynaTok仍能保持超过95%的基线准确率，超越了最近的无需训练方法。这些结果表明，DynaTok为高效和鲁棒的视频推理提供了原则性的基础，为未来Video-LLMs的实时流媒体视频理解铺平了道路。

## Abstract
Recent advances in Video Large Language Models (Video-LLMs) have greatly expanded multimodal reasoning capabilities. However, the massive number of visual tokens extracted from long video sequences incurs prohibitive computational costs, limiting their deployment in real-world scenarios. Existing training-free token compression methods select tokens based on attention magnitude as a proxy for semantic importance, but often overlook positional bias and rely only on short-term temporal locality, leading to redundant spatio-temporal coverage and inefficient token usage. We present DynaTok, a training-free, temporally adaptive and bias-aware token compression framework that allocates token budgets across both temporal and spatial dimensions. Through a lightweight exponential moving average (EMA) memory, the Temporal Budget Allocation (TBA) module dynamically assigns fewer tokens to redundant frames and more to novel frames, capturing long-term temporal variation. The Spatial Budget Allocation (SBA) module complements this by selecting spatially diverse and semantically important features using activation-based attention maps, while leveraging a spatial memory to reduce redundancy from previously selected regions and mitigate positional bias. DynaTok integrates seamlessly with existing Video-LLMs such as LLaVA-OneVision and LLaVA-Video without retraining, and effectively preserves semantic coverage under aggressive compression. Experiments on four representative VideoQA benchmarks-MVBench, LongVideoBench, MLVU, and VideoMME-show that DynaTok retains over 95% of baseline accuracy even with a 90% token reduction, surpassing recent training-free approaches. These results demonstrate that DynaTok provides a principled foundation for efficient and robust video reasoning, paving the way toward real-time streaming video understanding with future Video-LLMs.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **问题背景**：视频大语言模型（Video-LLMs）需要将视频帧编码为大量视觉令牌，令牌数量成百上千，导致自注意力计算复杂度呈二次增长，推理成本极高，难以在资源受限场景或实时流媒体中部署。
- **现有方法的不足**：当前无需训练的令牌压缩方法（如 FastV、VisionZip）通常以注意力幅值作为语义重要度的代理进行令牌筛选，但存在两个严重缺陷：
  1. **位置偏置**（Positional bias）：视觉编码器在跨模态联合训练中继承了文本的模式，导致某些固定空间区域（如图像边界或角落）无论语义内容如何都会获得异常高的注意力分数，单纯基于注意力幅值会反复选择这些偏置区域，造成冗余。
  2. **短期时间局部性**：现有方法只在相邻帧的局部窗口内利用时间相似性压缩令牌，无法建模长时间跨度的帧间新颖度变化，在长视频或流式场景中容易保留大量冗余帧。
- **整体含义**：DynaTok 提出一种**无训练、时间自适应且偏置感知**的令牌压缩框架，通过动态分配时空令牌预算，从根本上解决上述两个问题，从而在极端压缩下仍保持高语义覆盖和推理精度。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
DynaTok 是一个两阶段流水线，给定视频令牌 \(X \in \mathbb{R}^{T \times N \times D}\)（T 帧，每帧 N 个令牌，维度 D）和全局保留率 \(R\)。

### 阶段一：时间预算分配（Temporal Budget Allocation, TBA）
- **核心思想**：通过指数移动平均（EMA）维护一个全局时间记忆 \(M_g\)，估计每帧与长期时间动态之间的差异（帧新颖度），对新颖度高的帧分配更多令牌，冗余帧分配更少。
- **具体步骤**：
  - 对每帧计算全局池化特征并与记忆计算 L2 差异：\(\Delta f_g = \| \text{GlobalPool}(X_t) - M_g \|_2\)。
  - 更新记忆：\(M_g \leftarrow (1-\alpha)M_g + \alpha \cdot \text{GlobalPool}(X_t)\)，其中 \(\alpha\) 为平滑因子。
  - 将各帧的差异归一化得到帧权重 \(w_t\)，分配预算：\(B_t = w_t \cdot T \cdot N \cdot R\)。

### 阶段二：空间预算分配（Spatial Budget Allocation, SBA）
- **核心思想**：利用基于激活的注意力图（而非 LLM 内部注意力矩阵）度量令牌语义重要性；通过分块分配预算和空间记忆惩罚冗余区域，促进空间多样性并缓解位置偏置。
- **具体步骤**：
  1. **激活注意力图**：计算 \(A_t = \text{norm}\left( \sum_{D} |X_t| \right)\)，作为模型无关的语义显著度。
  2. **分块重要性**：将每帧划分为 P 个非重叠块，每个块的重要性 \(s_i = \text{mean}(A_t[p_i])\)，按比例分配每个块的令牌数量 \(n_i = \frac{s_i}{\sum_j s_j} \times B_t\)。
  3. **冗余惩罚**：维护空间记忆 \(M_s\) 存储先前帧选中的令牌。对每个令牌 x 计算与记忆的余弦相似度 \(\Delta f_s[x] = \text{cosine}(x, M_s[x])\)。
  4. **综合评分**：\(\text{score}(x) = A_t[x] - \beta \Delta f_s[x]\)，其中 \(\beta\) 控制冗余惩罚强度。
  5. 在每个块内选择得分最高的 \(n_i\) 个令牌，更新空间记忆：\(M_s[K_t] \leftarrow (1-\alpha)M_s[K_t] + \alpha X_t[K_t]\)。

### 关键特性
- **无需训练**，即插即用。
- **兼容 FlashAttention**，不依赖内部注意力矩阵，仅用视觉编码器的激活值。
- **两阶段联合优化**，同时解决时间冗余和空间偏置。
