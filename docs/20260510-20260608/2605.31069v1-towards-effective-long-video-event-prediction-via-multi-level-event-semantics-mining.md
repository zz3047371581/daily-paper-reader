## 摘要
准确预测未来事件是跨领域内容理解和决策的基础。虽然先前研究主要聚焦于文本或短视频场景，但长视频事件预测以海量多模态上下文和更复杂叙事为特征，仍待深入探索。同时，尽管近期基于大语言模型（LLMs）和视觉-语言模型（VLMs）构建的长视频语言模型（LVLMs）在长视频问答和摘要方面表现出潜力，但它们难以泛化到事件预测，因为既无法精确提取事件相关细节，也不能对事件发展进行细粒度分析。为解决这一差距，我们提出 VISTA，一个用于长视频事件预测的多层次事件语义挖掘框架。首先，VISTA 应用以角色为中心的视觉提示，精确提取事件相关的视觉细节，增强细节层语义；随后，采用知识增强的迭代检索策略，引导大语言模型逐步构建逻辑连贯的事件链，从而提升事件层叙事；最终，VISTA 采用类似人类的“先提议后检索”策略，生成多样化的未来导向提议并整合多层次线索，产生稳健而准确的预测。在真实世界数据集上的广泛实验验证了 VISTA 对长视频事件预测的有效性。

## Abstract
Accurately predicting future events is fundamental to content understanding and decision-making across various domains. While prior research has primarily focused on text or short-video scenarios, long-video event prediction, characterized by vast multimodal context and more complex narratives, remains underexplored. Meanwhile, although recent Long-Video Language Models (LVLMs), built on Large Language Models (LLMs) and Vision-Language Models (VLMs), have shown promise in long-video question answering and summarization, they struggle to generalize to event prediction, as they can neither precisely extract event-related details nor perform fine-grained analysis of event development. To address this gap, we propose VISTA, a multi-level event semantics mining framework for long-video event prediction. Initially, VISTA applies a character-centric visual prompt to precisely extract event-related visual details, enhancing detail-level semantics; subsequently, it employs a knowledge-enhanced iterative retrieval strategy, guiding the LLM to progressively construct logically coherent event chains, thereby improving event-level narratives; ultimately, VISTA adopts a human-like propose-then-retrieve strategy to generate diverse future-oriented proposals and integrate multi-level clues, producing robust and accurate predictions. Extensive experiments on real-world datasets validate the effectiveness of VISTA for long-video event prediction.

- **研究背景**：未来事件预测（FEP）在文本和短视频领域已取得进展，但长视频（如电影、电视剧）因包含海量多模态上下文和复杂叙事结构，仍是有待解决的空白地带。  
- **现有局限**：尽管基于大语言模型（LLM）和视觉‑语言模型（VLM）的长视频语言模型（LVLMs）在问答、摘要等任务中表现良好，它们在事件预测上性能急剧下降。根本原因在于  
  - **视觉细节的语义偏差**：现有 VLM 偏向低层视觉元素（如物体、服饰），忽视人物身份、动作等高层事件线索；  
  - **事件演进粗粒度**：LVLMs 只能给出高度概括的摘要，缺少对事件发展逻辑的细粒度分析，导致事件级语义大量损失。  
- **研究目标**：提出一种能精确挖掘细节‑事件‑预测多层次语义的框架 VISTA，以实现在长视频中稳健且可解释的未来事件预测。

## 2. 方法论详解
VISTA 框架通过三个模块层层递进，实现多级语义挖掘：

### 2.1 多模态预处理
- 利用 **WhisperX** 将音频转录为对话文本 \(T\)；  
- 利用 **PySceneDetect** 识别场景边界，将原视频切分为片段 \(S_i\)，并拼接为不少于 3 分钟的片段 \(C_i\)。

### 2.2 Describer：角色中心视觉提示
**目标**：精确提取事件相关的视觉细节，减弱背景噪声。  
**流程**：  
1. 使用 **InsightFace** 在每帧中检测人脸边界框，通过 **CLIP** 提取人脸嵌入；  
2. 对给定的角色肖像同样提取嵌入，计算余弦相似度，为每个人脸匹配最佳角色；  
3. 在视频帧上用不同颜色标注角色边界框，并配以文本提示（如 “Truman（蓝色）”），构成 **角色中心视觉提示**；  
4. 将带提示的帧输入 VLM，生成聚焦角色身份和动作的视觉描述 \(vd\)。

### 2.3 Narrator：知识增强迭代检索
**目标**：从时序事件中构建逻辑连贯的事件链，增强事件级叙事。  
**流程**：  
1. 对每个片段 \(C\)，将对话 \(D_C\) 和视觉描述 \(vd_C\) 输入 LLM，生成按时序排列的事件描述序列 \(\{E_{C_j}\}\)；  
2. 采用 **迭代式处理**：将当前事件 \(E\) 视为待处理，之前的事件视为历史事件 \(\{H_i\}\)；  
3. 引入基于 **ATOMIC 常识知识图谱** 微调的 **Flan‑T5** 作为常识专家 \(f_{\text{CE}}\)，为每个历史事件生成潜在的因果结果 \(\{O_{H_i}^j\}\)；  
4. 将历史事件与其因果结果拼接后嵌入，与当前事件嵌入 \(e_E\) 计算余弦相似度，选取 **top‑k** 最相似的历史事件作为候选；  
5. 将候选事件所在的事件链、因果指导及当前事件一起输入 LLM，判断当前事件应归入哪条事件链；  
6. 重复直至所有事件被整合，形成覆盖全片的多条事件链。  
**公式示例**：  
\[
\text{sim}(e_{H_i}^j, e_E)=\frac{e_{H_i}^j \cdot e_E}{\|e_{H_i}^j\|\|e_E\|},\quad e_{H_i}^j=f_t(H_i \oplus O_{H_i}^j)
\]

### 2.4 Predictor：先提议后检索
**目标**：基于事件链和多级线索进行可靠预测。  
**流程**：  
1. **提议阶段**：将每条事件链输入 LLM，生成多条未来可能的走向（\(P_k\)）；  
2. **检索阶段**：  
   - 事件级线索：将每条提议嵌入 \(e_{P_k}\) 与之前历史事件‑因果结果对嵌入计算相似度，用阈值 \(\tau_e\) 筛选相关历史事件；  
   - 细节级线索：在事件级线索对应的时间范围内，直接计算事件原始嵌入与所含细节的相似度，用阈值 \(\tau_d\) 筛选视觉/对话细节；  
3. **融合预测**：将每条提议和它对应的多级线索一起输入 LLM，选择最合理的提议并进一步细化为最终预测 \(\hat{E}\)。

## 3. 实验设计
### 3.1 数据集
- **CMD**（Condensed Movies 测试集改编）：4987 个事件‑视频对，平均时长 48.6 分钟，基于电影并做匿名化防泄漏。  
- **MILES**：从 2025 年后发布的 80 部电视剧中人工标注 624 个样本，平均时长 78.5 分钟，双重审核确保质量。  
- 两数据集均提供 IMDb 角色肖像作为先验知识。

### 3.2 评估指标
- **生成式评估**：ROUGE‑L (R)、METEOR (M)、CIDEr (C)、Sentence‑BERT (SB)。

## 6. 主要结论与发现
- **性能大幅领先**：VISTA 在 CMD 上所有指标绝对提升至少 8.14%（R）~17.49%（Acc），在 MILES 上至少 7.95%（R）~18.10%（Acc），准确率均超 60%。  
- **跨体裁鲁棒**：无论电影紧凑叙事还是电视剧松散结构，VISTA 均稳定高效，展现良好泛化性。  
- **多层次语义不可或缺**：缺少视觉细节会导致灾难性下降，缺少事件链会损害因果推理能力。  
- **三个核心组件均显著有效**：CVP 显著改善视觉细节提取，KIR 强化事件因果组织，PtR 使预测更精准且可解释。  
- **对 LLM 骨干不敏感**：更换不同开源、商用 LLM 均能保持竞争力，仅随模型能力提升而性能上升。  
- **关键超参数存在适宜区间**，但框架对参数变化不敏感，体现鲁棒性。

## 7. 优点
- **层次化语义挖掘**：首次将长视频事件语义明确区分为细节、事件、预测三层，逐级增强，设计合理。  
- **类人认知策略**：“先提议后检索”与人类推理过程类似，提升了可解释性，且所有中间过程可追踪、可分析。  
- **常识增强的因果推理**：利用外部知识图谱将无结构的事件转化为因果链，有效弥补单纯 LLM 的因果推理短板。  
- **角色中心视觉提示**：通过细粒度特征匹配和颜色标注，将 VLM 的注意力精准导向人物动作，避免无关背景干扰。  
- **全面实验验证**：双数据集、多指标、大量对比方法及详尽的消融与超参数分析，结论扎实。
