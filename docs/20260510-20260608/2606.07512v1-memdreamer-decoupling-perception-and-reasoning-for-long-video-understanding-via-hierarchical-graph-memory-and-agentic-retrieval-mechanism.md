## 摘要
当前的视觉-语言模型在处理长达数小时的视频时面临困难，因为处理全长的视觉序列会导致难以承受的令牌爆炸和注意力稀释。为了克服这一问题，我们引入了MemDreamer，将感知与推理解耦，将长视频理解转化为一个智能体探索过程。作为一个即插即用的框架，它逐步流式传输视频以构建层次图记忆，这是一种自顶向下的三层架构，用于语义抽象，并以捕获时空和因果关系的基础图为锚点。在推理过程中，推理模型利用智能体工具增强检索，通过观察-推理-行动循环导航层次结构、搜索节点并遍历逻辑边。实验表明，MemDreamer在四个主流基准上取得了SOTA结果，与人类专家的差距缩小至仅3.7个点。它将推理上下文窗口限制为全上下文摄入量的仅2%，同时提供了12.5个点的绝对准确率增益。此外，统计分析揭示了VLM在逻辑推理和长视频理解基准上的表现之间存在强正线性相关性，确立了智能体能力扩展作为多模态理解的新范式。

## Abstract
Current Vision-Language Models struggle with hours-long videos because processing full-length visual sequences induces prohibitive token explosion and attention dilution. To overcome this, we introduce MemDreamer to decouple perception and reasoning, shifting long-video understanding into an agentic exploration process. As a plug-and-play framework, it incrementally streams videos to construct a Hierarchical Graph Memory, a top-down three-tier architecture for semantic abstraction, anchored by a foundational graph capturing spatiotemporal and causal relations. During inference, the reasoning model employs agentic tool-augmented retrieval, navigating hierarchies, searching nodes, and traversing logical edges via an Observation-Reason-Action loop. Experiments show MemDreamer achieves SOTA results across four mainstream benchmarks, narrowing the gap with human experts to only 3.7 points. It constrains the reasoning context window to merely 2% of full-context ingestion while delivering a 12.5 point absolute accuracy gain. Furthermore, statistical analysis uncovers a strong positive linear correlation between an VLM's performance on logic reasoning and long-video understanding benchmarks, establishing agentic capability scaling as a new paradigm for multimodal comprehension.

- MemDreamer 在四个长视频基准上**全面达到 SOTA**，在 LVBench 上最高**90.7 分**，与人类专家（94.4）仅差 3.7 分。

### 7. 优点：方法或实验设计上的亮点

- **解耦范式**：从根本上避免了超长视频的令牌爆炸和注意力稀释，且可插拔不同感知/推理模型。
- **层次图记忆**：模拟人类从粗到细的理解方式，同时保留实体间时空因果拓扑，为检索提供强结构化先验。
- **通用工具增强检索**：不止步于相似度匹配，而是通过层级导航、语义/时间搜索、图遍历三大类工具实现可解释多步推理。
- **统计相关性揭示新方向**：用 Pearson 相关证明了推理能力与长视频理解的强链接，为未来研究提供了优化靶点（提升 Agentic 能力）。
- **丰富的消融实验**：从记忆结构、检索策略到超参数扫描，全面量化各组件贡献，结果稳健且解释清晰。
- **极致的上下文压缩**：推理窗口仅 5‑6 千令牌，使模型甚至可在低上下文环境下完成小时级视频问答。

### 8. 不足与局限

- **纯文本记忆的信息损失**：所有视觉细节被抽象为文本描述和图结构，对于依赖精确视觉外观（如着装、微表情）或需要查看原始画面才能回答的问题，可能存在信息丢失；附录案例中也显示，依赖 OCR 覆盖的文字来弥补，但并非所有视觉信息都能文本化。
- **构建记忆的成本未完全量化**：虽然感知模型只需处理 10 分钟窗口，但处理整个长视频仍需大量 API 调用或 GPU 资源，论文未具体报告时间开销和经济成本。
- **实时性与流式构建**：记忆构建是离线过程，对于实时或低延迟推理场景可能不适用，且依赖一个强大的感知模型。
- **层次与图的本体论设计依赖于人工定义**：节点类型、边类型和聚类规则由提示驱动，可能在某些开放式或非编剧型视频（如监控、体育直播）中泛化性不足；论文主要在烹饪、新闻、影视类基准上测试。
- **消融实验中未分析不同感知模型对记忆质量的影响**：虽然推理模型鲁棒性实验显示感知模型的差距被平滑，但未单独分析感知模型自身能力（如定位、描述错误）如何影响最终性能。
- **代理循环的最大步数有限**：12 步限制可能对特别复杂的问题不够，但实验显示平均仅使用约 3 步，故大部分情况足够，极端情况可能不足以完成深度多跳推理。
- **未与某些最新的极长上下文模型（如 Gemini 3.1 原生 1M 上下文）进行更深度的上下文效率对比**（有比较得分但未讨论如果给端到端同样的计算/上下文预算时是否还会提升）。
