## 摘要
在长时自中心视频中回答导航相关问题，需要检索和整合分布在远距离时间片段中的证据，同时保持空间和上下文一致性。尽管长上下文视觉-语言模型能够获得较强的回答质量，但它们在处理长轨迹时计算成本高昂，且对于重复查询效率低下。最近，基于图的方法（如 VL-KnG）通过持久化的时空知识图谱应对这一挑战，但仅依靠图中心检索可能无法充分表达更广泛的时间连续性和上下文线索。我们提出 VL-MemKnG，一个混合记忆框架，在 VL-KnG 的基础上结合了时空知识图谱和持久化的片段级上下文记忆。知识图谱捕捉结构化关系信息和长范围物体关联，而片段级记忆则保留更广泛的时间上下文，用于长跨度证据检索。混合检索与推理模块同时操作这两种记忆表示，生成基于证据的答案并按时序组织支持证据。我们还引入了 WalkieKnowledgeT+，这是 WalkieKnowledge 的扩展，用于面向长跨度导航的视频问答。该基准包含需要聚合来自多个非共现时刻的证据的时序分布推理任务。在 WalkieKnowledgeT+ 上，VL-MemKnG 将 Top-1 检索准确率从 58% 提升至 67%，Recall@1 从 34.50% 提升至 40.55%，优于所有对比方法，包括 Gemini 2.5 Pro 和 Qwen 3.5+。在时序全局和时序分散聚合问题上增益尤为显著，证明了在保持高效查询推理的同时，将结构化关系记忆与片段级上下文记忆相结合的优势。

## Abstract
Answering navigation-relevant questions over long egocentric videos requires retrieving and organizing evidence distributed across distant temporal moments while maintaining spatial and contextual consistency. Although long-context vision--language models can achieve strong answer quality, they are computationally expensive for long trajectories and inefficient for repeated querying. Recent graph-based approaches such as VL-KnG address this challenge through persistent spatio-temporal knowledge graphs, but graph-centric retrieval alone may underrepresent broader temporal continuity and contextual cues. We present VL-MemKnG, a hybrid memory framework that extends VL-KnG by combining a spatio-temporal knowledge graph with persistent segment-level contextual memory. The knowledge graph captures structured relational information and long-range object associations, while segment-level memory preserves broader temporal context for long-horizon evidence retrieval. A hybrid retrieval-and-reasoning module jointly operates over both memory representations to produce evidence-grounded answers and temporally organized supporting evidence. We also introduce WalkieKnowledgeT+, an extension of WalkieKnowledge for long-horizon navigation-oriented video question answering. The benchmark includes temporally distributed reasoning tasks requiring evidence aggregation across multiple non-cooccurring moments. On WalkieKnowledgeT+, VL-MemKnG improves Top-1 retrieval accuracy from 58% to 67% and Recall@1 from 34.50% to 40.55%, outperforming all compared methods, including Gemini 2.5 Pro and Qwen 3.5+. The gains are particularly pronounced on temporal-global and temporally scattered aggregation questions, demonstrating the benefits of combining structured relational memory with segment-level contextual memory while maintaining efficient query-time inference.

- **研究背景**：机器人在长时导航中积累的自中心视频轨迹蕴含丰富环境中观察信息，可用于后续决策，如定位目标位置、回忆访问历史。从轨迹中检索并推理回答自然语言问题，需跨远距时间段聚合证据并保持空间与上下文一致性。

  - 最近基于时空知识图谱的方法（如 VL‑KnG）构建持久结构化记忆，可高效检索，但对更广泛的时序上下文连续性和非结构化线索捕捉不足，在需要跨多个非连续时刻聚合证据的时间性问答上性能受限。
  - 现有视频记忆系统多为非结构化描述索引，缺乏对象关联和空间关系的显式表达。
- **研究目标**：提出一种**混合持久记忆框架**，同时融合时空知识图谱的结构化关系记忆与片段级上下文记忆，以在长时自中心轨迹上实现高效、可反复查询的时序证据定位与答案生成，并提升时间性推理任务的性能。

## 方法论
- **核心思想**：采用记忆中心设计，将计算密集的视频处理放在离线阶段；在线查询时并行检索知识图谱与片段记忆，融合证据后进行联合推理，输出答案与时间对齐的证据（平级或分组排名）。
- **框架流程（VL‑MemKnG）**：
  1. **离线记忆构建**：
     - **视频分段与描述生成**：将轨迹划分为等长时序片段（约 2 秒），每段采样关键帧，用 VLM 生成**对象中心化描述**（Prompt 约束输出对象、属性、空间关系、可读文本）。
     - **片段级记忆 `S`**：每个片段存储标识 `s_i`、描述 `c_i`、文本嵌入 `e_text_i`（text‑embedding‑3‑large）和视觉嵌入 `e_vis_i`（ViCLIP）形成索引。
     - **时空知识图谱 `G`**：从描述中抽取对象节点（含语义属性）和空间关系边，采用 VL‑KnG 的**时空对象关联 (STOA)** 机制链接跨块相同对象，构建全局图 `G = (O, R)`。
  2. **在线混合检索与推理**：
     - **检索阶段** `ϕ_retr`：
       - **片段记忆检索**：将问题转换为对象中心查询，计算查询文本、视觉嵌入与片段记忆的相似度，通过**加权融合**（默认 `w_vis = w_text = 0.5`）获得排序，截取 Top‑`k_mem` 片段集 `S_q`。
       - **知识图谱检索**：将问题改写为符合图中实体描述的查询，检索相关子图 `G_q`（图检索管线源自 VL‑KnG）。
     - **帧级证据映射**：将检索到的片段（含时间区间）映射到稀疏采样帧集合 `Ṽ` 中的具体帧索引，作为证据候选。
     - **推理阶段** `ϕ_ans`：将 `S_q` 的描述与帧、`G_q` 的对象/属性/空间关系证据输入大型语言模型，模型根据问题类型输出**平级排名**（单时刻证据）或**分组排名**（多非连续时刻证据），并预测最终答案。

- **关键技术与公式**：
  - 多模态融合检索得分：`s(i) = w_vis · s_vis(i) + w_text · s_text(i)`。
  - 支持两种证据结构：`F*(q)` 为平级帧序列 `[t_1,...,t_K]`，或分组证据 `{F*(1)_q,...,F*(M)_q}`，每组对应一个独立时刻的证据，适应时间全局和分散聚合问题。

## 主要结论与发现
- **整体性能提升**：在 WalkieKnowledgeT+ 上，VL‑MemKnG 将 **Top‑1 检索准确率从 VL‑KnG 的 58% 提高到 67%，Recall@1 从 34.50% 提高到 40.55%**，MRR@1 由 0.583 升至 0.668，均优于包括 Gemini 2.5 Pro 在内的所有对比方法；并保持了较高的答案准确率（65.81%）。
- **时间推理问题增益显著**：对时间全局和分散聚合问题，VL‑MemKnG 相比纯图 VL‑KnG 的 MRR 和 Recall 大幅提升，且在分散聚合问题上答案准确率居首，证明片段级上下文记忆有效弥补了图方法在时序连续性上的不足。
- **高效查询**：VL‑MemKnG 在查询延迟上远低于全上下文 VLM，且离线成本随查询次数增多而被摊薄，适合长轨迹多次查询场景。

## 优点
- **创新性的混合记忆**：首次将持久片段级上下文记忆与时空知识图谱结合，实现**结构化关系推理与宽域上下文捕获取长补短**，在时间推理任务中获得显著增益。
- **支持多证据格式**：框架可输出平级或分组证据排名，直接适配单时刻和时间聚合型问题，拓宽了问答系统的适用范围。
- **记忆中心架构**：计算密集型视频处理仅离线执行一次，在线查询轻量快速；记忆可复用，显著提高多查询场景的性价比。
- **实验严谨且透明**：提供丰富消融实验和类别分析，对比基线全面，并公开了新基准 WalkieKnowledgeT+，推动领域发展。
- **可解释性**：模型产出时间对齐的帧级证据，有利于机器人场景下的行为决策解释。

## 不足与局限
- **依赖描述生成质量**：片段记忆和知识图谱均源于视觉‑语言模型生成的对象中心化描述，描述偏差或遗漏会传播至检索和推理，且未探讨不同 VLM 对描述质量的影响。
- **离线图构建缺乏增量性**：知识图谱一次性离线生成，无法适应动态变化场景或在线增量更新，限制其在实际连续探索中的适用性。
- **基准规模与泛化性**：WalkieKnowledgeT+ 含 8 条轨迹、262 问题，规模有限；轨迹均来自 EgoWalk 数据集，环境与问题多样性需进一步验证。
- **内存构建成本透明度不足**：论文未清晰给出离线阶段的算力、时间和存储开销，仅展示了 token 成本平摊优势，不利于精确评估部署条件。
- **融合权重与检索深度敏感性**：消融实验显示视觉主导权重会显著降低性能，说明系统对多模态融合策略较为敏感；但所选默认超参表现稳定。
- **知识图谱的局限性**：图构建仍依赖固定空间关系词汇，可能遗漏复杂空间语义，且对象关联算法在遮挡或视角剧烈变化下可能出错。
