## 摘要
视觉-语言模型如CLIP可以为开放词汇目标检测提供丰富的语义先验。然而，将文本和视觉知识联合集成到检测架构中仍然具有挑战性。本文提出VL-DINO，一种开放词汇检测器，通过更有效地利用CLIP的视觉-语言知识来增强DINO。具体而言，首先开发了一个查询引导的正样本构建（QPSC）模块，以构建额外的高质量正样本，使vanilla DINO框架能够更好地适应异构数据源的混合训练，同时提供更多的视觉-语言对齐信号，从而在训练过程中融入更丰富的文本知识。随后引入了一个视觉语义编码器（VSE）模块，将CLIP视觉知识蒸馏到骨干网络提取的特征中，生成融合特征用于后续编码器精炼。基于融合特征，目标区域语义对齐（ORSA）模块提取以目标为中心的区域特征，并将其与相应的文本嵌入对齐，进一步融入文本线索。在零样本设置下，VL-DINO-T和VL-DINO-L在LVIS基准上分别达到36.3和38.1的AP，持续优于先前先进的方法。大量实验证明了所提出设计的有效性和竞争性能。

## Abstract
Vision-language models like CLIP can provide rich semantic priors for open-vocabulary object detection. However, jointly integrating both textual and visual knowledge into detection architectures remains challenging. In this paper, we propose VL-DINO, an open-vocabulary detector that enhances DINO through more effective exploitation of CLIP's vision-language knowledge. Specifically, a Query-guided Positive Sample Construction (QPSC) module is first developed to construct additional high-quality positive samples, enabling the vanilla DINO framework to better accommodate mixed training across heterogeneous data sources while providing more vision-language alignment signals, thereby incorporating richer textual knowledge during training. A Visual Semantic Encoder (VSE) module is then introduced to distill CLIP visual knowledge into backbone-extracted features, producing fused features for subsequent encoder refinement. Based on the fused features, an Object-Region Semantic Alignment (ORSA) module extracts object-centric region features and aligns them with the corresponding textual embeddings, further incorporating textual cues. In the zero-shot setting, VL-DINO-T and VL-DINO-L achieve 36.3 and 38.1 AP on the LVIS benchmark, respectively, consistently outperforming prior advanced approaches. Extensive experiments demonstrate the effectiveness and competitive performance of the proposed design.

- 现有方法在注入文本知识时依赖复杂跨模态交互模块，在利用视觉知识时受 CLIP 粗粒度全局对齐限制，且**两类知识难以在统一检测器中高效协同**。
- 整体含义：以 DINO（DETR-like 检测器）为基座，通过更精巧的视觉‑语言知识融合，实现强泛化、低复杂度的零样本目标检测。

### 方法论
VL-DINO 由三个核心模块构成，训练时引入，推理时仅保留 VSE，保持结构简洁。

#### 1. 查询引导的正样本构建（QPSC）
- **背景**：原始 DINO 的去噪机制需提前已知所有类标签，难以处理含自由形式短语的接地数据；且 DINO 的一对一匹配导致正样本稀疏。
- **思路**：受查询长尾分布观察启发，选取前 300 个内容查询（可靠查询池），对每个真实框做有界扰动生成辅助框（IoU>0.5），与随机采样的可靠内容查询组成额外正样本查询。
- **作用**：绕过匈牙利匹配，提供密集正监督；无额外可学习参数；提供更丰富的视觉‑语言对齐信号。

#### 2. 视觉语义编码器（VSE）
- **结构**：由 3 个 CLIP 风格自注意力块 + MLP 组成，作用于高阶段特征 P5、P6。输入添加 `CLS` token，输出 `CLS_l` 和增强的视觉标记。
- **蒸馏目标**：三部分损失
  - **全局语义对齐损失** `L_sem`：使 `CLS_l` 与 CLIP 的 `CLS_clip` 余弦相似。
  - **批内关系损失** `L_rel`：匹配批内样本间相似性分布（KL 散度）。
  - **稠密空间语义损失** `L_spa`：匹配补丁级 token 与类别嵌入的关系分布。
- **融合**：增强后的 P5、P6 与原始骨干特征残差相加，生成新的多尺度特征 `\~P5`, `\~P6`。

#### 3. 目标区域语义对齐（ORSA）
- 使用可学习查询 + 3 层解码器（仅分类对齐，无需回归），以真实框提取第一层编码器特征中的前景区域特征。
- 与对应文本嵌入通过**双向对比损失**（图像→文本 + 文本→图像）对齐，强化罕见类别的语义辨识。

### 主要结论与发现
- **零样本 LVIS 上最优**：VL‑DINO‑T 36.3 AP (38.7 Fixed AP)，VL‑DINO‑L 38.1 AP (40.2 Fixed AP)，全面超越现有方法，尤其稀有类提升显著。

- **模块协同有效**：QPSC 提供密集正样本与视觉‑语言对齐；VSE 的语义、关系、空间三重蒸馏是增益主因；ORSA 的对齐进一步提升了罕见类辨识。
- **迁移能力强**：在 COCO 线性探测/全微调下，VL‑DINO 表现大幅领先 YOLOE，体现高泛化性。

### 优点
- **模态融合简洁高效**：训练期间巧妙融入视觉和文本知识，推理时几乎不增加额外组件（仅保留 VSE）。
- **不依赖复杂融合**：相比 GLIP/GDINO，无需逐层交叉模态模块和超大规模训练数据，却取得更优性能。
- **正样本构建创新**：QPSC 无额外参数、不增加推理负担，有效利用查询分布特性。
- **多尺度语义蒸馏**：VSE 在不破坏低层细节特征前提下，为高层特征注入全局语义，且从图像级、批内关系、空间稠密三个层面进行知识传递。
- **实验扎实全面**：多基准、多尺度模型、丰富消融，对噪声鲁棒性和迁移性均有验证。
