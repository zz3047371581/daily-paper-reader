## 摘要
传统的图像描述方法往往难以生成全面、富含上下文的描述，尤其是对于无法直接从视觉线索中观察到的细节。为克服此问题，我们提出了一种新颖的检索增强图像描述框架，通过利用外部知识生成具有更深层次见解的描述，如物体属性、事件背景和潜在意义。我们的方法采用分层多模态文章检索机制，超越了单一的文本实体。该检索考虑了文章结构感知特征，包括加权的文本组成部分（例如标题、正文部分）和视觉布局模式，以及多方面的相似性计算（内容-视觉、视觉-视觉和话语定位）。随后的上下文相关性细化阶段进一步增强了检索到的信息。然后，检索到的文章作为描述生成的知识基础：首先，视觉语言模型生成简明的图像描述；其次，我们根据此描述从检索到的文章中分割出相关信息；最后，大语言模型利用描述和提取的知识生成全面、上下文详细的描述。我们参加了ACM Multimedia EVENTA 2025挑战赛，在OpenEvent-V1数据集的私有测试集上以0.2824的总分获得第五名。源代码已公开发布于https://github.com/mf0212/EVENTA-Challange。

## Abstract
Traditional image captioning methods often struggle to generate comprehensive, context-rich descriptions, especially for details not directly observable from visual cues. To overcome this, we propose a novel retrieval-augmented image captioning framework that generates captions with deeper insights, such as object attributes, event context, and underlying significance, by leveraging external knowledge. Our approach features a hierarchical multi-modal article retrieval mechanism that moves beyond monolithic text entities. This retrieval considers article structure-aware features, including weighted textual components (e.g., headlines, body sections) and visual placement patterns, alongside multi-faceted similarity computations (content--visual, visual--visual, and discourse positioning). A subsequent contextual relevance refinement stage further enhances the retrieved information. The retrieved articles then serve as the knowledge base for caption generation: first, a VLM generates a concise image description; second, we segment relevant information from the retrieved articles based on this description; and finally, an LLM utilizes both the description and extracted knowledge to generate a comprehensive, contextually detailed caption. We participated in the ACM Multimedia EVENTA 2025 Challenge and achieved 5th place with an overall score of 0.2824 on the private test set of the OpenEvent-V1 dataset. Source code is publicly released at https://github.com/mf0212/EVENTA-Challange.

### **方法论**

论文提出了一个名为**分层多模态检索增强的图像描述框架**，其运作分为两个核心阶段：

#### **1. 分层多模态文章检索**
该阶段旨在从庞大的文章数据库中，为输入图像找到最相关的新闻文章。其核心创新在于不仅考虑文本相似性，还深入挖掘文章的内部结构和多模态信号。

*   **核心理念**：将新闻文章视为结构化实体，而非单一文本块。
*   **关键技术细节**:
    *   **文章结构感知特征提取**: 将文章分解为标题(Headline)、导语(Lead)、正文(Body)和图像说明(Image Captions)等组件，并赋予它们不同的学习权重 ($\alpha_H, \alpha_L, \alpha_B, \alpha_C$)，因为标题等部分蕴含了更核心的语义。
    *   **视觉布局特征融合**: 提取文章内嵌图像的视觉特征 (fv(·))，并根据图像在文中的位置（例如是靠近标题还是结尾）进行加权调制 (g(·))，形成一个带有位置感知的视觉特征向量。
    *   **多层面相似性计算**: 计算查询图像与候选文章在三方面的相似度得分，并最终组合 ($score(q, a_i) = \beta_1 s_{\text{content}} + \beta_2 s_{\text{visual}} + \beta_3 s_{\text{position}}$)：
        1.  **内容-视觉对齐 ($s_{\text{content}}$)**：图像视觉特征与文章结构化的加权文本特征之间的余弦相似度。
        2.  **视觉-视觉一致性 ($s_{\text{visual}}$)**：查询图像与文章内嵌图像之间的最大余弦相似度。
        3.  **话语定位得分 ($s_{\text{position}}$)**：一个新颖的指标，认为图像在文章中的位置越关键（如首段），其与查询图像的相似度权重越高，这反映了事件中心性。
    *   **上下文相关性优化**: 在初步相似度计算后，引入**时间聚类（Temporal Clustering）**（增强同期相关文章）和**引用网络分析（Citation Network Analysis）**（PageRank算法提升有影响力的文章）来进一步优化检索分数。

#### **2. 上下文感知的图像描述生成**
该阶段将检索到的Top-1文章作为知识库，通过三阶段流程生成最终描述。

*   **核心理念**：将复杂任务分解为“视觉理解”、“知识提取”和“融合生成”三步，确保描述既忠于图像又富含事实。
*   **关键技术细节**:
    1.  **第一阶段：结构化视觉上下文提取**：使用视觉语言模型（VLM）作为“视觉分析员”，对输入图像 $I_q$ 生成一个结构化的视觉描述 $D_{\text{visual}}$，要求其包含客观描述、上下文推断、情绪氛围和潜在新闻标题四个维度的分析。
    2.  **第二阶段：相关上下文提取**：使用M3-Embedding (纯文本模型)将 $D_{\text{visual}}$ 和Top-1文章的每个句子进行向量嵌入，计算余弦相似度，选出Top-3最相似的句子，并扩展其前后相邻句子，经过重排序构成聚合证据 $A_{\text{retrieved}}$。选择纯文本模型而非多模态模型的原因是其在纯文本检索设定下效果更优。
    3.  **第三阶段：知识增强描述合成**：将输入图像 $I_q$、视觉上下文 $D_{\text{visual}}$和聚合的文本证据 $A_{\text{retrieved}}$ 一起提供给一个配置为“专家新闻图片说明撰写者”的大语言模型（LLM）。LLM被指示锚定于 $D_{\text{visual}}$ 中的视觉证据，同时使用 $A_{\text{retrieved}}$ 注入具体事实（姓名、地点、日期、事件意义），最终生成内容翔实的描述 $C_{\text{final}}$。

### **主要结论与发现**

1.  **多模态结构化检索至关重要**: 分解文章结构（标题、正文等）并赋予不同权重，比将文章视为一整块文本进行检索的效果更好。
2.  **视觉与位置信号是强大的相关性判别器**: 引入视觉-视觉一致性和基于文章话语位置的得分，能极大提升检索性能，验证了文章版面布局与事件中心性之间的强相关性。
3.  **三阶段描述生成流程有效**: 将“视觉理解”与“知识融合”解耦，先由VLM生成结构化的视觉“锚点”，再由LLM进行知识增强，有效防止了模型忽略图像内容、单纯总结文本的问题，生成了视觉忠实且信息丰富的描述。
4.  **语义相关性优于n-gram匹配**: 在官方测试中，尽管CIDEr得分不高，但CLIP Score很高，表明模型生成的描述虽然在词汇上与人工参考不完全一致，但在高层次语义上与图像高度相关，这在新闻描述这类开放性、事实性任务中是重要优势。
