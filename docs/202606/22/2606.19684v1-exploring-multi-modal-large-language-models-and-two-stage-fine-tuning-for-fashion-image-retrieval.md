## 摘要
组合图像检索使用由参考图像和修改文本描述组成的组合查询来检索目标图像。在时尚领域，该任务需要理解颜色、图案和纹理等细微的属性变化。然而，现有方法由于标注数据稀缺和简单的负采样策略而面临局限。我们提出了一种新颖的框架，该框架集成了多模态大语言模型（LLaVA）以生成属性感知的三元组，并引入两阶段微调策略来增强对比学习。我们利用预训练的视觉-语言模型，如CLIP-ViT/B32，生成句子级提示并将其与相对描述拼接，同时使用静态表征扩展负样本数量。实验结果展示了增强的组合推理能力和改进的细粒度检索行为，突显了所提框架在时尚检索中的可行性和潜力。

## Abstract
Composed image retrieval retrieves a target image using a composed query of a reference image and a modified text description. In the fashion domain, this task requires understanding subtle attribute variations such as color, pattern, and texture. However, existing approaches face limitations due to scarce annotated data and simplistic negative sampling. We propose a novel framework that integrates a multi-modal large language model (LLaVA) to generate attribute-aware triplets and introduces a two-stage fine-tuning strategy to enhance contrastive learning. We leverage pretrained vision-language models, such as CLIP-ViT/B32, to generate and concatenate sentence-level prompts with the relative caption and to scale the number of negatives using static representations. Experimental results demonstrate enhanced compositional reasoning and improved fine-grained retrieval behavior, underscoring the feasibility and potential of the proposed framework for fashion retrieval.

## 1. 研究背景与核心问题

时尚组合图像检索（Fashion Composed Image Retrieval, FIR）要求模型根据**参考图像**和**修改文本**（如“变成浅蓝色长袖碎花”）检索出符合修改意图的**目标图像**。这一任务高度依赖对颜色、图案、纹理、版型等**细粒度属性变化**的理解。

当前方法面临两大难题：
- **标注数据匮乏**：高复杂性、多属性的修改描述标注稀少，难以让对比学习充分捕捉细节差异。
- **简单负采样策略**：主流方法采用 **批次内随机负样本**，无法提供外观高度相似但属性微小不同的**难负样本**，导致模型难以学习精细的判别表征。

本文旨在**不依赖大量人工标注**的前提下，增强模型对细粒度时尚属性的感知能力和组合推理能力，从而提升检索精度。

## 2. 方法论

整体框架在标准组合图像检索双编码器结构（如 CLIP4CIR）上进行了三方面创新：

### 2.1 基于 LLaVA 的属性感知三元组生成
- 使用多模态大语言模型 **LLaVA** 为每张图像生成富含 **颜色、图案、纹理、风格** 的详细描述。
- 将参考图描述与原始修改文本拼接，再次输入 LLaVA 生成**上下文一致的修改描述**，形成（参考图描述，修改描述，目标图描述）三元组。
- 这种合成数据能捕获 CLIP 全局语义忽略的细粒度视觉属性，缓解标注稀缺。

### 2.2 混合负采样策略
为增强对比学习效果，设计三种负样本：
- **批次内负样本**：同批次其他样本的目标描述，提供基础多样性。
- **合成负样本**：对目标描述进行属性词替换（如将“淡蓝色”改为“翠绿色”），构造语义相似但不匹配的难负样本。
- **增强负样本**：交叉组合同一批次内不同三元组的元素，产生更加多样且难以区分的负样本。

### 2.3 两阶段微调
- **阶段一（粗对齐）**：同时训练查询编码器和目标编码器，使用批次内负样本，使模型初步对齐参考图、修改文本和目标图。
- **阶段二（精对齐）**：固定目标编码器，仅微调查询编码器，并引入难负样本（如前述合成或增强负样本），迫使模型聚焦于细微属性差异。

此外，查询端将 LLaVA 生成的参考图描述与用户修改文本拼接，形成上下文更丰富的文本输入，并利用预训练视觉‑语言模型生成句子级提示，强化组合推理。
