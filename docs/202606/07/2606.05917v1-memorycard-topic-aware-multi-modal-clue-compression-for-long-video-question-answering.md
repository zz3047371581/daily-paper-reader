## 摘要
长视频问答对视觉-语言模型仍具挑战，因为与答案相关的证据往往稀疏、瞬时且分散在冗长的视频上下文中。现有的以帧为中心的方法通过统一采样、查询感知帧选择、视觉标记压缩和自适应分辨率策略来提高效率。然而，它们仍以孤立且零散的帧作为基本证据单元，限制了视觉-语言模型有效捕捉连贯事件级语义的能力。为解决这一局限，我们提出MemoryCard，一个基于视频记忆的增强框架，将长视频组织为自包含的记忆卡片。具体而言，MemoryCard首先对视频及对齐的语音进行自读取过程，将视频分割成语义连贯的单元，每个单元对应一个不同的主题或事件。对于每个单元，它生成事件级视频要点并选取代表性的视觉时刻，进而渲染为统一的记忆卡片以供检索和问答。实验结果表明，在相当的视觉标记预算下，MemoryCard持续提升长视频问答性能，准确率相对提升最高达21.8%。所有代码已开源，见https://github.com/NEUIR/MemoryCard。

## Abstract
Long-video question answering remains challenging for Vision-Language Models (VLMs), as answer-relevant evidence is often sparse, transient, and temporally dispersed across lengthy video contexts. Existing frame-centric approaches improve efficiency through uniform sampling, query-aware frame selection, visual-token compression, and adaptive resolution strategies. However, they still rely on isolated and fragmented frames as the fundamental evidence units, limiting VLMs' ability to effectively capture coherent event-level semantics. To address this limitation, we propose MemoryCard, a video-memory-based augmentation framework that organizes long videos into self-contained Memory Cards. Specifically, MemoryCard first performs a self-reading process over videos and aligned utterances to segment the video into semantically coherent units, each corresponding to a distinct topic or event. For each unit, it generates an event-level video gist and selects representative visual moments, which are then rendered into unified Memory Cards for retrieval and question answering. Experimental results demonstrate that MemoryCard consistently improves long-video QA performance under comparable visual-token budgets, achieving up to a 21.8% relative improvement in accuracy. All code is available at https://github.com/NEUIR/MemoryCard.

## 2. 方法论

MemoryCard的核心思想是**将长视频组织为一组自包含的“记忆卡片”**，每张卡片对应一个语义连贯的事件/主题单元，融合事件级文本要点与代表性视觉瞬间。整体流程包含三个关键步骤：

1. **自读式语义单元构建**  
   使用一个VLM（如Qwen3-VL-8B）对完整视频及其对齐的语音转录进行“自读取”，将视频按主题、事件或场景转换分割为若干语义单元（session），每个单元带有起止时间戳和主题描述。  
   公式表达：  
   \[
   \{(b_i, e_i)\}_{i=1}^J = \text{VLM}(\text{Instruct}_{\text{seg}}, \mathbf{V})
   \]  
   其中 \(b_i, e_i\) 为第 \(i\) 个语义单元的开始与结束时间。

2. **记忆卡片渲染**  
   对每个语义单元 \(s_i = \mathbf{V}[b_i:e_i]\)：  
   - 生成**事件级视频要点** \(g_i\)，包含VLM产生的主题摘要和该时段内的关键语音文本；  
   - 根据 \(g_i\) 选出最能表征该单元的**代表性视觉帧** \(\tilde{\mathbf{F}}_{b_i:e_i}\)；  
   - 将代表性帧、要点文本及时间跨度渲染为一张**图像格式的记忆卡片** \(M_i\)：  
     \[
     M_i = \text{Render}(\tilde{\mathbf{F}}_{b_i:e_i}, g_i, (b_i, e_i))
     \]  
   最终得到记忆卡片库 \(\mathcal{M} = \{M_i\}_{i=1}^J\)。此过程不依赖下游问题，卡片可复用。

3. **问题感知检索与回答**  
   对给定问题 \(q\)，使用LongCLIP计算问题与每张卡片的相似度，选取Top-\(k\)张相关卡片 \(\tilde{\mathcal{M}}\)；  
   根据检索相关性将卡片分配至高、中、低分辨率；  
   按原始视频时间顺序重新排列后，送入回答VLM生成答案：  
   \[
   y = \text{VLM}(q, \tilde{\mathcal{M}})
   \]

## 6. 主要结论与发现

- MemoryCard在**可比的视觉标记预算**下，持续提升多款VLM在三个长视频问答基准上的性能，相对准确率提升最高达**21.8%**。  
- 性能增益来源于**高密度的多模态证据表示**，而非简单的检索：移除语音、主题或时间信息均会导致性能下降，证明渲染卡片内组件的有效性。  
- **事件感知的语义单元分割**比基于固定窗口或镜头的分割更能生成相干记忆卡片，对检索和回答更有利。  
- 在**保留精细视觉细节**（高分辨率卡片）与**扩大事件覆盖**（低分辨率卡片）之间，相关性驱动的分辨率分配能取得最佳平衡。  
- 记忆卡片按**原始视频时序排列**优于按检索得分或随机排列，表明时间结构对长视频推理的重要性。  
- 细粒度分析显示，MemoryCard在感知、识别、OCR和信息综述等需结合局部细节与事件语境的任务上增益明显，但在计数和推理任务上提升有限，提示当前方法仍需增强密集时间覆盖或多卡聚合能力。

## 7. 优点

- **方法论创新**：首次将视频证据从“帧”的层级提升至“事件级记忆卡片”，有效缓解长视频中语义稀疏和碎片化问题。  
- **兼容性强**：记忆卡片采用图像格式，自然兼容现有的图像VLM流程，无需修改模型架构。  
- **可复用与无需训练**：记忆卡片在问题不可知（question-agnostic）阶段预构建，可供同一视频的多个问题复用，且整个框架无需调整模型参数。  
- **实验扎实**：针对组件、单元构建方式、分辨率分配、排序策略等做了系统消融，清晰验证了各个设计选择的价值。  
- **结果显著**：在多个主干模型和数据集上取得一致提升，且增益并非仅由检索带来，而是由证据表示质量决定。

## 8. 不足与局限

- **预处理开销大**：需对每个视频运行VLM进行分割、摘要、选帧，并使用ASR提取语音，额外成本可能影响实时或大规模部署。  
- **对细粒度运动理解有限**：记忆卡片本质上是静态图像+文本，可能难以精细捕捉视频中的运动力学、连续动作或短暂时态，导致计数和部分动作理解任务提升较小。  
- **检索依赖风险**：采用“先检索后回答”的流水线，如果检索遗漏关键事件卡片，仍会导致答案错误，难以完全枚举长时程中的全部证据。  
- **视觉预算实验中**：分辨率组合固定，未探索更动态的分辨率分配策略；同时使用LongCLIP作为检索器，其跨模态对齐质量可能影响性能上界。  
- **未进行多模型检索器或更大规模VLM主干的对比**，可能限制了结论的通用性；也未测量实际推理延迟与资源占用。
