## 摘要
AI 眼镜为 AI 代理作为个性化记忆助理提供了一个引人注目的平台。要真正有用，此类系统必须超越短期视频理解，并解决人类在纵向自我中心视频流中因实际、个人或社交目的而遇到的记忆缺口。然而，现有的自我中心数据集主要侧重于动作识别或来自短片段的一般性问答，测量的是感知能力而非现实的人类记忆需求。我们引入了 SuperMemory-VQA，一个用于评估 AI 助理在实用、长时记忆任务上表现的自我中心视觉问答（VQA）数据集。它包含 52.9 小时的日常活动记录，由 AI 眼镜采集，包括同步的 RGB 视频、音频转录、眼动注视、IMU 和 SLAM 轨迹。通过人工验证的标注流程，我们构建了 4,853 个有依据的问答对，涵盖物体与位置记忆、意图回忆、视觉场景回忆、时间线重建、对话记忆和上下文检索。每个问题均为多项选择，并设有明确的“不可回答”选项以测试幻觉稳健性。对领先的代理框架和 LLM 主干的基准测试表明，现有系统在实际记忆任务上仍远未达到可靠水平，这凸显了需要新的架构来实现有依据的 AI 记忆，仅在证据充足时才做出回答。一项参与者调查进一步证实，我们的问题是现实、有用的，并与日常记忆需求相一致。

## Abstract
AI glasses present a compelling platform for AI agents to serve as personalized memory assistants. To be genuinely useful, such systems must move beyond short-term video comprehension and address memory gaps that humans experience for practical, personal, or social purposes over longitudinal egocentric video streams. However, existing egocentric datasets predominantly focus on action recognition or generic QAs from short clips, measuring perceptual capabilities rather than realistic human memory needs. We introduce SuperMemory-VQA, an egocentric visual question answering (VQA) dataset for evaluating AI assistants on practical, long-horizon memory tasks. It contains 52.9 hours of everyday activities recorded with AI glasses, including synchronized RGB video, audio transcription, eye gaze, IMU, and SLAM trajectories. Through a human-verified annotation pipeline, we construct grounded 4,853 question-answer pairs that span object and location memory, intent recall, visual scene recall, timeline reconstruction, conversational memory, and in-context retrieval. Each question is posed as multiple-choice with an explicit "unanswerable" option to test hallucination robustness. Benchmarking leading agentic frameworks and LLM backbones reveals that existing systems remain far from reliable on real-world memory tasks, highlighting the need for new architectures for grounded AI memory that can answer only when evidence is sufficient. A participant survey further supports that our questions are realistic, useful, and aligned with everyday memory needs.

        2.  **阶段二：代理式QA生成与验证**：

    *   **选择题设置**：每道题设有4个**有序选项**：`正确`、`模糊`（部分正确但模糊）、`错误`和`不可回答`（证据不足）。这旨在评估模型的**答案准确性**和**幻觉稳健性**。
    *   **证据基础**：每个答案都需由一段或多段跨时长的视频片段作为证据支持，要求模型进行**多证据推理**。

### **3. 实验设计**

*   **数据集与场景**：使用自建的 **SuperMemory-VQA** 数据集，包含52.9小时视频和4,853个问答对。
*   **基准方法与框架**：
    *   **框架**：评测了两种最先进的智能体框架：
        1.  **Video-RAG**：基于检索增强生成，通过检索关键帧、ASR、OCR等辅助文本来增强VLM的输入。
        2.  **EgoButler**：采用层次化记忆库，先进行粗到细的时间定位，再将检索到的片段描述输入VLM。此外还补充对比了**VideoAgent**。
    *   **视觉语言模型**：在这两个框架下，测试了多款**开源**（如Qwen-3-VL、InternVL-3.5、Gemma-4）与**闭源**（如Gemini-3-Flash/Pro、GPT-5.4/mini）VLM。
*   **评估指标**：
    *   `Ans-F1`：判断问题是否“可回答”的F1分数。
    *   `QA-Acc`：四选一多选题的准确率（`模糊`、`错误`和错误弃答均视为不正确）。
    *   `QA-MRR`：基于模型对答案选项排序的均值倒数排名，奖励将正确答案排在最前的行为。
*   **对比基线**：除主对比外，还设置了**纯文本盲测**（仅给LLM题目和选项，不给视频证据）来检验数据泄露风险。

### **6. 论文的主要结论与发现**

*   **现有系统远未可靠**：即使是最强的配置（Gemini-3-Flash + Video-RAG），QA准确率也仅为**61.0%**，表明长时记忆QA任务极具挑战性。
*   **检索框架的重要性**：Video-RAG在多数指标上优于EgoButler，尤其在**可回答性判断**上提升显著，说明结构化的证据检索比层次化的摘要式记忆更有效。
*   **模型能力与规模不单调**：闭源模型普遍优于开源模型，但模型性能**并非随规模单调提升**。例如，Gemini-3-Flash的表现优于更大的Gemini-3.1-Pro，因其更敢于在证据充分时作答，而后者过于保守；部分开源大模型甚至不如同系列小模型。
*   **主要失败模式**：最大问题是**过度弃答**。在面对可回答的问题时，许多模型（尤其是开源模型）错误地选择了“不可回答”，这表明模型的**认知校准能力**很差。
*   **定性分析揭示的挑战**：模型在精细证据检索、对话事实混淆、小物体/OCR识别、时间顺序追踪和前提验证等方面均存在显著缺陷。

### **7. 优点**

*   **问题定义精准务实**：数据集设计紧密围绕真实的人类记忆需求，定义了六大记忆任务，问题采用自然、生活化的措辞，实用价值高。
*   **评估维度全面**：通过有序选择题（正确/模糊/错误/不可回答）和多项评价指标（QA-Acc, Ans-F1, QA-MRR），不仅能评估答案对错，还能评估模型的**推理能力和幻觉稳健性**。
*   **基准测试系统严谨**：论文不仅报告了顶尖模型的表现，还通过盲测、多粒度分析、失败模式定性研究等手段，深刻剖析了现有系统的短板，为后续研究指明了方向。
*   **数据集质量高**：通过精心设计的多阶段人机协同标注流水线，并经过最终人工审核，确保了QA对的事实准确性和高质量。

### **8. 不足与局限**

*   **规模相对较小**：与Ego4D（3670小时）等感知数据集相比，52.9小时的数据量仍然偏小。10名参与者的样本量也可能限制了数据多样性的视角。
*   **场景泛化性有限**：数据主要在“模拟家居环境”中采集，且参与者按预设脚本活动。这与完全开放、无脚本的真实世界复杂性仍有差距。
*   **多模态信息利用不充分**：虽然数据集提供了眼动、IMU、SLAM等丰富模态，但基准测试中的两种框架（Video-RAG、EgoButler）主要利用的是视觉和音频（转录文本）信息，未能充分挖掘其他模态的潜在价值，所得结果不能视为利用全部传感器数据的上限。
*   **基准框架的局限性**：所选的智能体框架本身也是当前技术的代表，存在“长上下文丢失”等固有缺陷，这可能会影响对更先进架构潜在能力的评估。
