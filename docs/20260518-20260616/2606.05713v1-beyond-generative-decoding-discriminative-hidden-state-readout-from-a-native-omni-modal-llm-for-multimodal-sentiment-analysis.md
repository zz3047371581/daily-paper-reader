## 摘要
多模态情感分析（MSA）从语言、声学和视觉信号中推断人类情感。近期方法越来越多地通过生成式读出方式适配大规模多模态模型（LMM）：提示模型以文本字符串的形式输出情感得分。虽然方便，但这将连续回归与离散自回归解码绑定在一起，带来了未被测量的代价。我们重新审视这一读出机制，并提出了一种基于原生全模态大语言模型（Qwen2.5-Omni-7B）的Thinker模块的判别式构建方案。我们不再进行文本解码，而是通过一个轻量级回归头，在单次前向传播中将最后一个非填充token的末层隐藏状态映射为连续得分。使用4位量化和低秩适应（QLoRA），整个7B流程——包括视频和音频处理——在单张消费级GPU（RTX 5090，32 GB）上训练，峰值显存10-21 GB，可训练参数占比1.14%。通过固定骨干网络、数据和LoRA配置的受控比较，我们分离了读出机制的影响。在CMU-MOSI和CMU-MOSEI数据集上，我们的判别式读出在无需特定任务特征工程的情况下达到了最先进的准确率（MOSI: MAE 0.551, Corr 0.888; MOSEI: MAE 0.506, Corr 0.790），并展现出强大的多种子稳定性。相比之下，生成式读出——即使在同等监督训练之后——平均绝对误差增加了一倍以上，产生无法解析或超出范围的输出（零样本情况下达2.8%），并且延迟更高。模态消融实验揭示了在CMU-MOSI上文本主导的模式。我们的发现表明，LMM的读出方式与其训练方式同样重要，证明判别式读出为连续MSA提供了一个更准确、高效、可靠的替代方案。

## Abstract
Multimodal sentiment analysis (MSA) infers human affect from language, acoustic, and visual signals. Recent methods increasingly adapt large multimodal models (LMMs) via generative readout: prompting the model to emit a sentiment score as a text string. While convenient, this ties continuous regression to discrete autoregressive decoding, incurring unmeasured costs. We revisit this readout mechanism and propose a discriminative formulation built on the Thinker module of a native omni-modal LLM (Qwen2.5-Omni-7B). Instead of text decoding, we map the final-layer hidden state of the last non-padding token to a continuous score via a lightweight regression head in a single forward pass. Using 4-bit quantization and low-rank adaptation (QLoRA), the entire 7B pipeline -- including video and audio processing -- trains on a single consumer GPU (RTX 5090, 32 GB) with 10-21 GB peak memory and 1.14% trainable parameters. Through a controlled comparison fixing the backbone, data, and LoRA configuration, we isolate the impact of the readout. On CMU-MOSI and CMU-MOSEI, our discriminative readout reaches state-of-the-art accuracy without task-specific feature engineering (MOSI: MAE 0.551, Corr 0.888; MOSEI: MAE 0.506, Corr 0.790) and exhibits strong multi-seed stability. In contrast, the generative readout -- even after equivalent supervised training -- more than doubles the mean absolute error, yields unparsable or out-of-range outputs (2.8% zero-shot), and suffers from higher latency. Modality ablations reveal a text-dominant regime on CMU-MOSI. Our findings indicate that how an LMM is read out is as consequential as how it is trained, demonstrating that a discriminative readout offers a more accurate, efficient, and reliable alternative for continuous MSA.

## 1. 论文核心问题与整体含义
- **核心问题**：当前基于大规模多模态模型（LMM）的多模态情感分析（MSA）大多采用**生成式读出**（generative readout），即让模型自回归地解码出一个表示情感得分的文本字符串，再解析为数值。这种范式将连续回归任务强行绑定在离散的、自回归的解码通道上，但由此带来的精度损失、输出不可解析/越界、推理延迟等问题很少被系统量化。
- **研究动机**：揭示读出机制本身对连续 MSA 性能的决定性影响，而不是仅关注模型训练或特征工程。文章提出一个**判别式读出**（discriminative readout）方案，直接从模型隐藏状态回归连续情感值，而无需任何文本解码步骤。
- **整体含义**：对于连续强度回归型情感分析，**“怎么从模型读出答案”与“怎么训练模型”同等重要**，判别式读出在准确率、输出可靠性、推理效率上全面优于当前主流的生成式读出。

## 2. 方法论：核心思想与关键技术细节
### 2.1 判别式隐藏状态读出
- **骨干模型**：使用原生全模态 LLM **Qwen2.5-Omni-7B** 的 **Thinker** 模块作为共享编码器，丢弃原本用于语音生成的 Talker 头。Thinker 能统一处理文本、视频帧、音频（可选），通过因果自注意力将多模态信息融合为令牌序列的隐藏状态。
- **读出机制**：
  - 提取 **最后一个非填充令牌** 的最终层隐藏状态（维度 3584）作为聚合表示。通过掩码确定该位置，避免批量填充对表示的影响。
  - 该表示送入一个**轻量级 MLP 头**（线性层 → 层归一化 → ReLU → Dropout → 线性层），直接回归出一个标量情感得分 $\hat{y} \in [-3, +3]$。
  - 整个过程只需要**一次前向传播**，无解码、无采样、无解析，输出天然为有界实数，不会出现无法解析或越界的错误。
- **训练目标**：对训练集标签进行标准化（减均值除标准差）后，使用**L1 损失（MAE）**进行回归训练，并在评估时逆标准化。这提供了连续、尺度敏感的梯度信号。

### 2.2 生成式读出基线
- 与判别式读出共享完全相同的骨干、数据和 LoRA 适配，仅改变读出方式：
  - **训练**：将连续标签格式化为数字字符串（如 `1.50`），拼接到提示后，通过语言模型交叉熵损失进行训练（仅计算标签令牌的损失）。
  - **推理**：自回归地贪婪解码少量令牌，再用正则表达式解析出数值；若解析失败则记作“不可解析”，若数值超出 $[-3, 3]$ 则记作“越界”（并截断）。
- 该基线提供了**零样本**和**同等监督训练后**两个版本，确保比较的公平性。

### 2.3 参数高效适应：QLoRA
- 骨干权重冻�并量化为 **4-bit NormalFloat（NF4）**，计算时使用 BF16 精度。
- 在注意力机制的 q, k, v, o 投影以及前馈网络的 gate, up, down 投影中插入 **LoRA 适配器**（秩 $r=32$，缩放系数 $\alpha=32$，dropout 0.1）。只有 LoRA 矩阵和回归头需要训练，总共约 1.03 亿参数，占全模型约 **1.14%**。
- 使用梯度检查点、禁用 KV 缓存等技巧控制显存。

### 2.4 高效多模态输入管线
- 视频帧通过索引均匀采样（最多 16 帧），限制每帧像素预算，避免全片段解码，节省解码时间和显存。
- 使用子进程解码并设置超时机制，防止损坏视频卡死训练流程。
- 音频统一经过 **DeepFilterNet** 去噪预处理。

## 3. 实验设计

  - 回归指标：平均绝对误差（MAE）、皮尔逊相关系数（Corr）。

- **对比方法**：涵盖多模态融合的经典方法和前沿模型，包括 TFN、LMF、MFN、MulT、MISA、MAG-BERT、Self-MM、MMIM、ConFEDE、DMD、ALMT、MSAmba、MEMMI、DecAlign 等。经典基线数值引自 MMSA 统一基准[20]，近期方法数值引自各原论文。

## 6. 主要结论与发现
- **读出方式是决定性设计要素**：在完全相同的模型骨架和训练条件下，判别式读出在 MOSI 上的 MAE 仅为 0.667，而生成式读出零样本时为 1.443，训练后反而升至 1.521（Corr 亦劣化）。生成式读出出现约 2.8% 的不可解析输出，但判别式天然无此缺陷。
- **判别式读出可达 SOTA 性能**：在 MOSI 上取得 **MAE 0.551、Corr 0.888**，MOSEI 上 **MAE 0.506、Corr 0.790**，与最新的专用融合方法持平甚至部分指标更好，却无需手工特征提取和定制融合模块。
- **训练生成式读出的局限性**：额外的监督训练未能弥合差距，甚至会降低其相关性。原因是文本生成损失主要优化字符串格式而非情感幅度，而判别头直接接收尺度敏感的回归梯度。
- **模态贡献因数据集而异**：在 CMU-MOSI 上，文本是主导模态，加入视频和音频反而略微增加回归误差，但对二分类准确度略有提升。这反映了该数据的文本强信息特性，不宜过度推广。
- **表示空间具有良好的情感梯度**：隐藏状态的 PCA 和 t-SNE 投影清晰呈现从负到正的情感流形，近邻标签差异远低于随机基线，解释了为何简单的线性/MLP 头能有效读出情感。

## 7. 优点
- **首次对读出方式做严格受控对比**：系统量化了生成式读出在连续 MSA 中的精度、可靠性与速度代价，弥补了该领域的重要空白。
- **方法简单高效**：移除生成头、仅用最后 token 表示加 MLP 回归，显著降低了实现复杂度，同时提升了所有关键指标。
- **低硬件门槛、可复现**：在单张 RTX 5090 上即可完成 7B 全模态模型的训练与部署，显著降低了 LMM 应用于 MSA 的硬件壁垒。
- **回归设定规范**：标签标准化、L1 损失、基于掩码选 token 等细节设计保证了方法在非平衡、歪斜标注上的鲁棒性。
- **分析全面**：不仅报告性能，还包含模态贡献、稳定性、表示空间分析及对生成式读出失败模式的拆解。

## 8. 不足与局限
- **种子稳定性验证不全面**：多种子分析只在 MOSI 上进行，较大的 MOSEI 仅做了单次运行。
- **消融实验覆盖有限**：模态消融和音频去噪消融均只针对 MOSI，MOSEI 上未进行，可能遗漏跨数据集的模态差异。
- **基线比较存在潜在偏差**：性能对比中引用了来自不同研究、不同特征提取管道下的经典模型结果，并未在统一评估框架内复现所有基线，可能引入细微的不公平。
- **任务和语言范围窄**：所有实验均针对英语连续情感强度回归，是否适用于离散情感分类、多语言或跨文化场景尚未验证。
- **对文本主导数据集依赖明显**：在 MOSI 上多模态的增益不明显甚至轻微有害，但该发现是数据集依赖的，不能直接断言非语言模态无用，需在更多样化的数据上进一步确认。
