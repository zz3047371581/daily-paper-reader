## 摘要
心理构念通常通过不同的工具、数据集和研究传统来测量，这使得直接比较变得困难。本文提出一个框架，通过将这些构念表示为共享词嵌入空间中的方向并进行比较，使它们在语义上可通约。使用监督语义差异法，我们从文本-结果关联中估计特定于构念的语义梯度，并将其投影到理论驱动的参考轴上。作为初步测试案例，我们使用效价、唤醒度和支配度（VAD）作为情感坐标系统。首先，我们从英语词汇层面的情感规范中恢复了可解释的 VAD 方向。其次，将 27 个 GoEmotions 类别的语义梯度投影到该空间中，恢复了预期的情绪组织，尤其是在效价和唤醒度方面。第三，我们将相同的程序应用于源自 IPIP-NEO-300 项目-因素关联的大五人格领域和侧面。领域层面的布局总体上是连贯的，而侧面层面的结果则更具探索性，因为它们依赖于稀疏的问卷文本。结果表明，嵌入空间可以支持在原本不可通约的心理测量之间进行构念层面的比较，前提是语义定位的稳定性和可解释性得到评估。

## Abstract
Psychological constructs are often measured in separate instruments, datasets, and research traditions, which makes direct comparison difficult. This paper proposes a framework for making such constructs semantically commensurate by representing and comparing them as directions in a shared word-embedding space. Using Supervised Semantic Differential, we estimate construct-specific semantic gradients from text-outcome associations and project them onto theoretically motivated reference axes. As an initial test case, we use Valence, Arousal, and Dominance (VAD) as an affective coordinate system. First, we recover interpretable VAD directions from English word-level affective norms. Second, we project semantic gradients for 27 GoEmotions categories into this space and recover the expected organization of emotions, especially along valence and arousal. Third, we apply the same procedure to Big Five personality domains and facets derived from IPIP-NEO-300 item-factor associations. Domain-level placements are broadly coherent, while facet-level results are more exploratory because they rely on sparse questionnaire text. The results suggest that embedding spaces can support construct-level comparison across otherwise incommensurable psychological measurements, provided that semantic placements are assessed for stability and interpretability.

### 2. 方法论：核心思想与关键技术细节
- **核心思想**: 任何心理构念都能被表示为同一词嵌入空间中的一个方向（向量），通过比较这些方向的几何关系（如投影、相似度）来比较构念。
- **技术流程**:
    1.  **文本表示**: 使用 SIF 加权和主成分移除的 GloVe 词向量，将每个文本（如单词或问卷项目）转换为文档向量 `xi`。
    2.  **监督语义微分 (SSD)**: 对每个心理构念，将文档向量 `xi` 作为自变量，将与该构念关联的连续结果变量 `yi`（如情绪评分、项目-因子载荷）作为因变量，通过 PCA 降维后拟合一个线性模型：`yi = α + β⊤ ˜xi + εi`。

    4.  **跨构念比较**: 将所有构念的语义梯度向量，通过**余弦相似度**投影到一个预先定义好的、可解释的**参考坐标系**（本文为效价-唤醒度-支配度, VAD）上，从而获得每个构念在情感空间中的坐标。

### 6. 主要结论与发现
- **参考轴可恢复**: VAD三个情感维度在GloVe空间中均可恢复为清晰、语义可解释的方向，其中效价的恢复最为稳健（r=0.73）。
- **情绪组织符合理论**: 27种GoEmotions类别在VAD空间中的投影坐标与情感环状模型预测高度一致：正向情绪在效价正极，负向高唤醒情绪（如恐惧、愤怒）在效价负极和唤醒度正极。支配度与效价高度相关，但去除效价影响后，愤怒类情绪比恐惧类情绪表现出更高的支配度剩余。
- **人格定位语义连贯**: 大五领域在VAD空间中的位置具有理论可解释性：尽责性（高V，低A），神经质（低V，高A），外向性（高A）。人格侧面结果更具探索性，主要受限于项目文本稀疏。
- **框架可行性**: 共享词嵌入空间能够作为一种基础设施，支持对来自不同测量体系的心理构念进行语义层面的比较。

### 7. 优点与亮点
- **新颖的框架**: 提出了一个将“心理构念”作为基本分析单元，在共享语义空间中进行“测量”和比较的方法，超越了传统的单词或文本分析。
- **方法的可解释性**: 通过将抽象的语义方向投影到VAD等理论轴，结合最近邻词汇审计，提供了高度的可解释性和可视化便利性。
- **跨测量传统的整合**: 成功地将来自情绪分类、人格问卷两种完全不同测量传统的构念，映射到了同一个参考空间中并进行了比较，初步证明了方法的整合潜力。

### 8. 不足与局限
- **文本语义的局限**: 方法测量的是构念的**操作化语言**的语义，而非脱离语言的、纯粹的心理构念本身。例如，问卷项目的措辞会直接影响结果（如含有负面内容的项目会拉低效价），这一点 Sympathy（共情）侧面的坐标已有所体现。
- **侧面结果不稳定**: 由于每个大五侧面仅由10个项目构成，模型拟合的统计效力不足，结果具有探索性，稳定性存疑。
- **语料库与模型偏差**: GloVe 嵌入本身固有的局限（如缺乏语境、含有训练语料的社会偏见）会直接影响结果。例如，Openness（开放性）的信号大量与政治语料关联，Extraversion（外向性）的负极端则被程序性副词占据。
- **参考轴的普遍性**: VAD 提供的是一个有效但低维的参考框架，它无法穷尽所有心理意义。效价与支配度的高度相关表明这些轴的独立性有限。
- **样本限制**: 所有数据均来自英语母语者，且人格样本（IPIP-NEO-300）存在过度代表受过教育的西方人群的风险，结论的跨语言、跨文化泛化能力未知。
