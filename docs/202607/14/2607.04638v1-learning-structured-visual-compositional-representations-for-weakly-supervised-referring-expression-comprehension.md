## 摘要
指代表达理解（REC）旨在根据自然语言描述定位图像中的目标。在弱监督REC（WREC）中，现有方法主要依赖于锚点级别的视觉表征。即使辅以辅助线索，关系交互仍被隐式编码在单个锚点特征中，导致视觉表征保持扁平且仅包含一元信息，限制了其与语言结构化特性对齐的能力。本文提出了一个面向WREC的结构化视觉组合表征（SVCR）学习框架。所提出的SVCR不再将关系隐式编码在一元锚点中，而是显式地对一元目标嵌入和成对关系嵌入进行建模，形成一个结构化的视觉表征空间。我们进一步引入了一个组合对齐机制，以统一的方式将一元和成对视觉表征与相应的文本嵌入匹配，从而在弱监督下实现组合式视觉-文本匹配。在RefCOCO、RefCOCO+和RefCOCOg上的大量实验表明，所提SVCR达到了最先进的性能。这些结果证明了显式结构化视觉表征和视觉-文本对齐对于WREC的有效性。

## Abstract
Referring expression comprehension (REC) aims to localize the object in an image described by natural language. In Weakly supervised REC (WREC), existing approaches primarily operate on anchor-level visual representations. Even when enriched with auxiliary cues, relational interactions remain implicitly encoded within individual anchor features. The resulting visual representation remains flat and unary-only, limiting its ability to align with the structured nature of language. In this work, we propose a Structured Visual Compositional Representation (SVCR) learning framework for WREC. Rather than implicitly encoding relations within unary anchors, the proposed SVCR explicitly models both unary object embeddings and pairwise relational embeddings, forming a structured visual representation space. We further introduce a compositional alignment mechanism that matches unary and pairwise visual representations with their corresponding textual embeddings in a unified manner, enabling compositional visual-textual matching under weak supervision. Extensive experiments on RefCOCO, RefCOCO+, and RefCOCOg show that the proposed SVCR achieves state-of-the-art performance. These results demonstrate the effectiveness of explicit structured visual representations and visual-textual alignment for WREC.

## 1. 论文的核心问题与整体含义

- **研究背景**：指代表达理解（REC）需要根据自然语言描述在图像中定位目标。为降低人工标注成本，弱监督REC（WREC）仅利用图像–文本对进行训练，通过对比学习建立锚点与语言的对应关系。
- **现有问题**：当前WREC方法普遍基于**扁平的单锚视觉表示**，即使通过辅助线索（如基础模型特征、空间坐标）或关系消息传递增强了锚点特征，**对象间的关系最终还是被吸收进单独的一元嵌入中**，导致视觉表示失去结构化信息。
- **核心含义**：本文认为该“扁平化”是制约WREC性能的根本瓶颈，因为指代表达具有**组合性和关系性**（如“放在白色袋子上的黑椅子”）。为此首次明确提出**结构化视觉组合表征（SVCR）**，显式保持一元对象与成对关系这两个视觉单元，并在弱监督下实现**组合式视觉—文本对齐**，从而有效对齐语言的结构化特性。

## 2. 方法论

### 核心思想
- 把视觉表示空间拆分为**一元对象表征（UOR）**和**成对关系表征（PRR）**两个互补结构，替代以往单一的扁平锚点。
- 设计**组合对齐机制（CA）**：用主语名词块的文本嵌入对齐一元视觉表征，用全句嵌入对齐成对关系表征，再通过层次一致性约束统一两种粒度的证据。

### 关键技术细节（文字流程）

1. **特征提取**
   - **视觉编码器**：从预训练YOLOv3检测头取出前K个高置信度锚点特征 A = {a₁,…,a_K}，并使用动态路径聚合来自DINOv2、Depth Anything等基础模型的辅助视觉特征，得到丰富锚点 Ã。
   - **文本编码器**：LSTM + 自注意力处理指代表达，同时输出两个嵌入：全句嵌入 s 和主语名词块嵌入 s_c（例如“穿黑衬衫微笑的男人”中的“微笑的男人”）。

2. **一元对象表征（UOR）**
   - 引入N个可学习的全局语义基向量 P = {p₁,…,p_N}。
   - 每个锚点投影到语义基上获得语义组合 a_i^una，再与原始锚点通过门控融合生成最终的一元表征 â_i = g_i⊙a_i^una + (1-g_i)⊙ã_i。
   - 为促进语义基的多样性，施加正交约束损失 L_div。

3. **成对关系表征（PRR）**
   - 对于每个锚点对 (i,j)，利用语义特征和几何特征（相对位置、尺度比、IoU、角度等）编码，并引入自适应权重 w_ij 抑制噪声交互。
   - 输出显式保留的关系表征 r_ij，**不同之处是不再将关系信息再聚合回单个锚点**，从而维持结构化。

4. **组合对齐与训练**
   - **一元对齐**：计算 â_i 与 s_c 的相似度，用InfoNCE损失 L_una 优化，伪正样本由弱监督下置信度最高的锚点确定。
   - **成对对齐**：计算 r_ij 与 s 的相似度，同样用对比损失 L_pair 优化。
   - **层次一致性**：约束成对关系得分聚焦于在UOR下排名靠前的锚点，损失 L_cons。
   - **总损失**：L = L_una + L_pair + L_div + L_cons。

5. **推理**
   - 分别计算一元置信度（â_i^T s_c）和关系置信度（对称聚合 r_ij 与 s 的相似度），加权求和选出最佳锚点，解码其边界框作为最终定位结果。

## 6. 主要结论与发现

- 显式建模**结构化视觉表征**（UOR+PRR）并采用**组合对齐**，能够有效解决WREC中“扁平锚点”与结构化语言之间的失配问题。
- 该方法在RefCOCO、RefCOCO+和RefCOCOg上均达到**最先进的弱监督性能**，在YOLOv5主干下显著缩小了与全监督方法的差距（例如RefCOCO val上达到74.51%）。

- 提出的结构化框架**具有泛化性**，可直接集成到其他WREC方法（如WeakMCN）中并带来增益。
