## 摘要
情感感知的艺术图像生成要求图像与输入提示匹配、遵循指定的艺术风格并传达目标情感。该挑战的主要难点在于，训练数据中可用的视觉和情感属性在测试时并未明确提供。缺乏这些属性，生成器不仅要决定描绘什么，还要决定如何通过色彩、光照、笔触、构图、线条和布局来表达目标情感。这就造成了可用测试提示与情感感知艺术生成所需的细粒度条件之间的控制差距。为弥合这一差距，我们提出了EmoStyle，一个基于Z-Image的框架，将输入提示转换为结构化的生成状态。首先，LLM推理器预测情感线索（效价-唤醒度、主导情感和治疗效果标签）以及宽高比决策。我们没有将这些预测仅用作额外的提示文本，而是将情感字段编码为情感条件向量，并通过AdaLN风格的调制将其注入到去噪块中，从而让推断出的控制变量直接指导中间特征的生成。由于情感表达也与风格相关，我们进一步为每种艺术风格桶训练专用的LoRA适配器，并在推理时选择相应的专家，使得相同的情感线索能够借助不同风格桶特有的色彩、纹理、笔触和构图先验进行渲染。最后，一个轻量级VLM引导的候选选择步骤根据提示对齐度、风格一致性、情感表达和视觉质量对生成的图像进行排序。在AffectiveArt Challenge 2026的Track 1中，我们的USTC_PI_LAB_TEAM提交取得了第一名。

## Abstract
Emotion-aware artistic image generation requires an image to match the input prompt, follow the specified artistic style, and convey the target emotion. In this challenge, the main difficulty is that the visual and affective attributes available in the training data are not explicitly provided at test time. Without these attributes, the generator has to decide not only what to depict, but also how the target emotion should be expressed through color, lighting, brushwork, composition, line, and layout. This creates a control gap between the available test prompt and the fine-grained conditions needed for emotion-aware artistic generation. To bridge this gap, we propose EmoStyle, a Z-Image-based framework that converts the input prompt into a structured generation state. An LLM reasoner first predicts affective cues (valence-arousal, dominant emotion, and therapeutic-effect labels) and an aspect-ratio decision. Instead of using these predictions only as additional prompt text, we encode the affective fields into an affective condition vector and inject it into the denoising blocks through AdaLN-style modulation. This allows the inferred control variables to directly guide the generation of intermediate features. Since emotional expression is also style-dependent, we further train a dedicated LoRA adapter for each artistic style bucket and select the corresponding expert during inference, enabling the same affective cues to be rendered with bucket-specific priors for color, texture, brushwork, and composition. Finally, a lightweight VLM-guided candidate selection step ranks the generated images based on prompt alignment, style consistency, emotional expression, and visual quality. In Track 1 of the AffectiveArt Challenge 2026, our USTC\_PI\_LAB\_TEAM submission achieved first place.

### 1. 论文的核心问题与整体含义
- **核心问题：** 情感感知的艺术图像生成面临“控制缺口”。训练数据（EmoArt）提供了丰富的视觉属性与情感标注（如效价、唤醒度、主导情感、治疗效果标签、笔触、色彩、构图等），但测试时只给出一条简短文本提示和目标风格桶，这些细粒度条件缺失。
- **研究动机：** 现有文本到图像生成模型无法仅通过文本提示精确控制“如何通过色彩、光影、笔触等视觉形式表达特定情感”，因为提示中未显式包含这些结构化条件。此外，情感表达高度依赖艺术风格，不同风格即使传达相同情感，其视觉呈现也应截然不同。
- **整体含义：** 该工作试图通过从提示中**推断隐式情感属性**，并将其编码为**可直接注入扩散模型的特征条件**，再结合**风格专家的先验**来实现情感与风格的可控解耦生成，从而填补测试时的信息缺失，解决情感艺术生成的“什么内容 + 什么风格 + 表达何种情感”三合一难题。

### 2. 论文提出的方法论
- **整体框架（EmoStyle）**：基于 Z-Image 扩散模型构建 4 阶段流水线。
- **关键步骤与核心思想**：
  - **1. 提示到情感规划（LLM Reasoner）**
    - 输入：用户提示 \(x_i\) 和风格桶 \(y_i\)。
    - 用 Gemini-3.5-Flash 生成结构化**情感计划** \(P_i\)，包含：连续效价-唤醒度分数 \((\nu_i,\eta_i)\)、主导情感 \(e_i\)、治疗效果标签集 \(H_i\) 以及长宽比 \(\rho_i\)。
    - 这样做而非改写提示，避免依赖不稳定的自然语言属性。
  - **2. 风格专家选择与推理**
    - 对每个风格桶（如水墨、浮世绘、文艺复兴等 8 种）在冻结 Z-Image 上单独训练一个 **LoRA 适配器**（秩 128）。
    - 测试时根据 GT 风格桶**确定性映射**到对应的风格 LoRA 专家，避免训练不稳定的路由器，并使风格控制完全可解释。
  - **3. 情感条件调制（Affective Modulation）**
    - 将通过推理得到的情感字段编码为**情感条件向量** \(c_{i,\text{aff}}\)：
      - 效价-唤醒度用正弦坐标嵌入保留连续性；
      - 离散情感与治疗标签用可学习嵌入；
      - 组合为拼接特征并加入交互项（连续坐标嵌入与主导情感标签嵌入的元素积），再经共享编码器 \(g_\phi\) 得到最终向量。
    - **注入方式**：使用 **AdaLN 风格的残差调制**，为每个风格桶配备一个残差调制模块 \(M_{l,y_i}\)。该模块以时间步嵌入和情感条件向量为输入，预测对 Z-Image 原生缩放-门控参数的**偏移量** \(\Delta s, \Delta g\)，从而在保留原生权重的前提下，实现情感信息对去噪过程的干预。
    - 训练时仅优化情感编码器、调制模块和风格 LoRA，Z-Image 主干冻结。
  - **4. VLM 引导的候选精炼**
    - 生成多张候选图（不同随机种子），用 **Qwen3.5-397B-A17B** 作为评判者，从提示一致性、风格一致性、情感计划一致性、无参考视觉质量四个维度打分，选出最优候选。
    - 该步骤无需额外训练，仅靠测试时扩展提升结果。

### 6. 论文的主要结论与发现
- EmoStyle 在 **AffectiveArt Challenge 2026 Track 1 中获得第一名**，总体评分显著优于其他方法。
- 在本地验证上，完整框架相比 Z-Image 基线的 **FID 降低 22.7%，AAS 提升 7.6%**。
- **组件贡献明显**：长宽比预测对降低 FID 最关键，情感规划、风格 LoRA 和候选精炼均对属性与风格对齐有正向提升。
- 结构化情感条件注入与风格专家解耦的设计，有效填补了测试时信息缺失造成的控制缺口，实现了内容、风格、情感的三重一致。

### 7. 优点
- **控制解耦设计**：明确将“风格选择”（LoRA 专家）与“情感调制”（条件向量）分离，使系统更可解释、易扩展。
- **结构化条件注入**：没有简单地将推理结果作为文本拼接，而是通过编码和 AdaLN 残差调制直接干预特征，使情感控制更难被文本忽略。
- **强知识与先验利用**：LLM 提供从简略文本到细粒度情感属性推理的常识，VLM 提供多维度质量判断，两者结合降低了对训练数据直接标注的依赖。
- **轻量化训练**：仅更新 LoRA 适配器、情感编码器和调制头，保持基座模型冻结，训练开销可控。
- **实际表现突出**：在挑战赛中夺冠，验证了方法的实战有效性。

### 8. 不足与局限
- **实验覆盖窄**：仅在 EmoArt 一个数据集上进行实验，缺少在其他情感图像数据集（如 FI、ArtEmis）或更通用的文本到图像生成基准上的泛化验证。
- **对 LLM/VLM 的依赖与偏差**：情感规划依赖 Gemini，候选排序依赖 Qwen，这两个模型的内部偏好可能引入系统性偏差（例如对某些情感或风格的偏好），且推理成本未分析。
- **风格桶受训练数据限制**：需要为每种风格单独训练 LoRA，当风格数量大幅增多时，存储和训练成本会线性增长；Gongbi 等小样本风格还需额外数据补全。
- **宽高比预测的合理性**：长宽比由 LLM 离散预测，可能与实际艺术规范不完全吻合，生成图像可能出现非标准的比例。
- **情感表达的深层评估缺失**：评价依赖机器指标和 VLM 打分，未包含人类主观评价或艺术家对情感传达准确性的专业判断，可能无法完全反映“艺术性”和“情感共鸣”。
- **训练细节未公开**：缺乏训练算力、耗时等资源报告，复现和实际应用的成本难以评估。
