## 摘要
视频异常检测对于自动化监控至关重要，但仅依赖可见光视频时，在光照变化、快速运动和复杂背景等挑战性条件下仍表现脆弱。为解决这些限制，我们提出了EVAD，一个事件增强的视频异常检测框架，它联合利用传统视频和由仿生事件相机捕获的事件流。事件传感器以高时间分辨率异步捕获亮度变化，对运动模糊和极端光照具有鲁棒性，并提供与基于视频的视觉信息互补的运动显著性线索。为支持多模态视频异常检测研究，我们构建了一个大规模的可见光-事件基准数据集，包含在不同光照水平、运动模式和背景复杂度下采集的63亿事件和376,368帧视频，填补了用于事件异常检测的现实可扩展数据集的空白。在此数据集基础上，我们设计了一个对比多模态预训练框架，通过对齐事件流、可见视频和文本描述之间的语义嵌入，学习具有判别力的事件表示。然后，一个自适应融合模块动态整合事件的时间线索与视频的空间语义，提高对环境干扰的鲁棒性。在基准数据集和我们提出的TJUTCM-Pha数据集上的实验表明，EVAD始终优于其他方法，验证了基于事件的感知在真实场景视频异常检测中的有效性。

## Abstract
Video anomaly detection (VAD) is critical for automated surveillance but remains fragile under challenging conditions such as illumination variations, fast motion, and complex backgrounds when relying solely on visible light videos. To address these limitations, we propose EVAD, an event enhanced VAD framework that jointly exploits conventional video and event streams captured by bio inspired event cameras. Event sensors asynchronously capture brightness changes with high temporal resolution, offering robustness to motion blur and extreme lighting, and providing motion salient cues complementary to video based visual information. To support multi modal VAD research, we construct a large scale visible event benchmark comprising 6.3 billion events and 376,368 video frames collected under diverse illumination levels, motion patterns, and background complexities, filling the gap of realistic and scalable datasets for event based anomaly detection. Building upon this dataset, we design a contrastive multi modal pretraining framework to learn discriminative event representations by aligning semantic embeddings across event streams, visible videos, and textual descriptions. An adaptive fusion module then dynamically integrates event based temporal cues with video based spatial semantics, improving robustness to environmental disturbances. Experiments on benchmarks and the proposed TJUTCM Pha dataset demonstrate that E VAD consistently outperforms methods, validating the effectiveness of event-based sensing for VAD in real world scenarios.

## 2. 方法论 
**核心思想**：将事件流与视频视为互补的感知模态，通过对齐事件、视频、文本三种模态的语义空间，再以自适应融合方式协同利用时序运动和空间语义，最终实现弱监督下的片段级异常评分。

**关键技术细节**  
- **事件表示**：将异步事件流按固定时间窗累积成“事件帧” \(E_i\)，使其能够被标准图像编码器处理。
- **多模态预训练模块**  
  - 输入编码：用CLIP ViT‑L/14分别提取视频特征 \(\hat{F}_v\)、事件特征 \(\hat{F}_e\)（经可学习投影映射至视频空间）及由ConceptNet扩展的文本特征 \(F_{ot}\)。  
  - 时序建模：用Transformer‑like结构融合全局注意力与局部掩码注意力，通过可学习参数μ门控式聚合长期与短期时序信息，得到 \(F_{oe}\) 和 \(F_{ov}\)。  
  - 对比学习：构造事件‑文本（\(L_{et}\)）、视频‑文本（\(L_{vt}\)）、视频‑事件（\(L_{ve}\)）三个InfoNCE损失，将三种模态拉入同一语义空间。总预训练目标 \(L_{total\_pre} = \theta L_{et} + \beta L_{ve} + \gamma L_{vt}\)。
- **自适应融合模块（AWF）**：用一个轻量线性层习得门控参数 \(\alpha = \sigma(W_\alpha[\hat{F}_v; \hat{F}_e] + b_\alpha)\)，以 \(\alpha \odot \hat{F}_v + (1-\alpha) \odot \hat{F}_e\) 的方式动态加权融合，再经投影得到融合特征 \(F_{out}\)。轻量线性门控可避免在长尾小样本VAD任务中过拟合。
- **片段级异常评分**：对融合的片段特征 \(F_{out}\) 使用因果卷积 \+ sigmoid 输出异常分数 \(s_i\)。
- **总损失**：在弱监督下，用 top‑k 片段平均作为视频级预测，计算二分类交叉熵 \(L_{cs}\)，同时加入视觉‑文本一致性KL散度正则项 \(L_{kd}\)，最终 \(L = L_{cs} + \lambda L_{kd}\)。

## 7. 优点
- **感知范式创新**：首次将真实事件流作为独立感知模态引入VAD，跳出传统“视频+衍生辅助信息”的框架，从感知层面解决鲁棒性问题。  
- **首个真实事件‑视频VAD基准**：TJUTCM Pha填补了领域空白，为后续研究提供高价值平台。  
- **统一多模态对齐**：通过视频‑事件‑文本三模态对比预训练，将事件表示嵌入语义空间，提升了可解释性与泛化性。  
- **轻量高效融合**：AWF以极简线性门控实现动态加权，避免过拟合且效果出色。  
- **实验极其全面**：从多个数据集、多类消融、细粒度定性分析等多维度验证了方法的有效性。

## 8. 不足与局限
- **真实事件数据覆盖有限**：除TJUTCM Pha外，另外两个基准的事件数据为屏幕翻拍或仿真，与真实事件传感器特性存在差异，方法在多样化真实事件场景下的泛化性尚需更多真实数据集检验。
