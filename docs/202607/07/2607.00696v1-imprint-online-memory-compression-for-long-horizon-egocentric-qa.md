## 摘要
长时程自我中心问答涉及回答数小时或数天前发生的事件。这要求记忆表征在数天或数周的记录中既保持检索有效性又具有可扩展性。现有的长时程自我中心问答方法将记忆构建为观察的分层文本摘要。虽然这能有效减少记忆大小，但摘要优化的是描述性压缩而非检索：重复的交互被吸收进粗略的文本描述中，而不是被保留为显式的、重复出现的记忆单元，这使得长时程证据聚合变得困难。我们提出Imprint，一个以交互为中心的记忆框架，将长时程自我中心记忆形式化为一个在线记忆压缩问题，而非摘要问题。传入的观察首先被表示为结构化的交互记录，并持续组织成重复出现的交互模式。利用人类记忆巩固中的重复性、近期性和独特性信号，Imprint选择性地保留并压缩交互，形成紧凑的、面向检索的记忆。我们在EgoLifeQA上评估了Imprint，这是一个包含七天自我中心数据的基准测试，其中的问题需要对查询前数小时到数天发生的交互进行推理。在使用相同大语言模型的情况下，Imprint将问答准确率从31.0%提升至35.8%，相比于EgoRAG使证据支持的答案增加了6倍，内存占用减少了2.3倍，检索延迟降低了11.8倍。这些结果表明，记忆压缩为长时程自我中心问答提供了可扩展且检索有效的基础。

## Abstract
Long-horizon egocentric question answering involves answering about events that have occurred hours or days in the past. This requires memory representations that remain both retrieval-effective and scalable over days or weeks of recording. Existing long-horizon egocentric QA methods construct memory as hierarchical textual summaries of observations. While effective for reducing memory size, summarization optimizes for descriptive compression rather than retrieval: repeated interactions are absorbed into coarse textual descriptions instead of being preserved as explicit, recurring memory units, making long-horizon evidence aggregation difficult. We propose Imprint, an interaction-centric memory framework that formulates long-horizon egocentric memory as an online memory compression problem rather than summarization. Incoming observations are first represented as structured Interaction Records and continuously organized into recurring interaction patterns. Using human memory consolidation signals of recurrence, recency, and distinctiveness, Imprint selectively retains and compresses interactions into a compact retrieval-oriented memory. We evaluate Imprint on EgoLifeQA, a seven-day egocentric benchmark containing questions that require reasoning over interactions occurring hours to days before the query. With the same LLM, Imprint improves QA accuracy from 31.0% to 35.8%, increases evidence-grounded answers by $6\times$ compared with EgoRAG, reduces memory footprint by $2.3\times$, and decreases retrieval latency by $11.8\times$. These results demonstrate that memory compression provides a scalable and retrieval-effective foundation for long-horizon egocentric question answering.

- **现有方法局限**：主流方法（如EgoRAG）先将视频转化为非结构化字幕，再构建分层文本摘要。虽然摘要能减少存储，但其优化的是**描述性压缩**，而非**检索有效性**；重复出现的交互会被吸收为粗糙的文本描述，难以保留结构化的重复记忆单元，导致难以进行跨时间证据聚合（如“我最近是否更常不吃午餐？”）。
- **论文意义**：提出Imprint框架，将长时程自我中心记忆构建为一个**在线记忆压缩**问题，而非摘要问题，用认知启发的信号保留面向检索的交互结构。

### 2. 论文提出的方法论
- **核心思想**：以**交互记录**（Interaction Record）为中心，每条记录包含人、动作、对象及起止时间戳；对交互记录进行在线压缩，在保证可扩展性的同时保留可检索的细粒度结构。
- **关键技术细节**：
  - **交互记录提取**：用LLM（Qwen2.5-7B-Instruct）从每段视频字幕中解析出结构化交互记录。
  - **在线记忆压缩（三阶段）**：
    1. **事件原型分配**：将交互记录编码后，与事件原型库比较，分配至最近的原型（相似度超阈值），或创建新原型（容量限制$N_{max}$）。原型维持重复次数和最近时间戳。
    2. **交互重要性评分**：利用人类记忆巩固的**频率、近期性、独特性**信号，为每条记录计算重要性分数：
       $$w(f_t) = \underbrace{\log(1+\nu_{n_t})}_{\text{frequency}} \cdot \underbrace{e^{-\lambda \Delta T}}_{\text{recency}} \cdot \underbrace{\frac{1}{1+\log(1+N_t)}}_{\text{distinctiveness}}$$
       分数超过阈值$\tau$的交互被保留。
    3. **交互巩固**：对保留的交互记录，若其动作-对象表示及上下文化嵌入与已有记忆条目相似，则合并（event consolidation），保留时间顺序和重要性；否则新建记忆条目。最终形成紧凑的压缩记忆$M$。
  - **检索与问答**：给定问题，提取对象、动作、时间约束等线索，对候选记忆计算排序分数（实体相似度、动作相似度和重要性分数的加权和），取top-$k$，必要时按时间过滤，最后交给LLM生成答案。
- **算法流程**：采用**查询感知字幕**（用64个元问题引导字幕生成）→ 交互记录提取 → 在线压缩 → 检索 → LLM回答的流线。

### 6. 论文的主要结论与发现
- 交互为中心的结构化记忆比分类文本摘要**更能支持长时程证据聚合**，同一LLM下，Imprint将QA准确率从31.0%提升至35.8%，证据支撑率从10.8%大幅提升至64.8%（6倍）。

- 在线压缩实现了**更低的存储（减少2.3倍）和更快的检索（加速11.8倍）**，同时处理开销与摘要法相近。
- 对**习惯洞察类问题**的提升尤为明显，说明Imprint能更好地保留并聚合重复行为模式。

### 7. 优点
- **创新性问题转化**：将“摘要”重新定义为“在线记忆压缩”，更贴近长时程检索的需求。
- **认知理论驱动**：引入人类记忆巩固中的重复性、近期性和独特性信号，设计简单且可解释的重要性分数。
- **结构化交互表征**：交互记录（人-动作-对象）保留细粒度语义，显著提升证据支撑，避免大模型仅靠先验猜答案。
- **高效且可扩展**：随着记录天数增加，内存增长平缓，且检索延迟极低，适合持续记录场景。
- **实验扎实**：引入Grounded Accuracy，设计多维度消融，对比公平，并开放了超参数分析。

### 8. 不足与局限
- **依赖字幕质量**：整体性能受限于视频字幕的质量，如果人、物或地点的识别有误，错误会向下游传播。
- **重要性评分基于启发式**：当前频率、近期性、独特性权重由超参数固定，未进行学习适配，可能无法适应个体行为差异性；论文也提到未来可探索可学习的记忆启发式。
- **任务类型局限**：对**TaskMaster（任务规划、多步推理）** 的提升相对较小，说明单纯优化记忆并不足以解决需要复杂推理的问题。
- **单人评估**：实验仅在EgoLifeQA的单个参与者（Jake）上进行，对其他设备、更复杂的人际交互或不同行为模式的泛化性有待验证。
- **离线扩展性未深入**：虽然在线压缩实时性尚可，但对于海量长期部署（数月以上）的压缩效率和检索性能仍有待进一步检验。
