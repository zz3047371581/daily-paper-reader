## 摘要
视频推理分割要求在复杂的自然语言查询下，对数百帧进行像素精度的目标跟踪，产生密集的时空令牌，其二次自注意力成本使得长视频处理难以实现。现有方法通过令牌压缩来解决，但通常作用于缺乏时序上下文的编码器特征，从而在能够可靠评估内容冗余之前就限制了选择。明智的压缩需要上下文感知，但以全分辨率获取这种感知会带来压缩旨在减少的相同的二次成本。状态空间模型解决了这一限制，因为其线性循环以 O(T) 的成本选择性地将每个令牌基于时序上下文进行调节，产生可评估内容冗余的表示。在此基础上，选择性时空聚合与压缩 (STAC) 通过解耦的双向空间和因果时间扫描丰富特征，利用循环导出的冗余进行分层压缩，并采用根据分割目标优化的自适应阈值。STAC 在零样本流式兼容设置下，实现了 85% 的令牌减少和 1.8 倍的加速，同时在推理分割基准上超越了无压缩的基线方法。代码见此处。

## Abstract
Video reasoning segmentation demands pixel-accurate object tracking across hundreds of frames under complex natural language queries, producing dense spatiotemporal tokens whose quadratic self-attention cost makes long-video processing prohibitive. Existing methods address this through token compression, yet typically operate on encoder features lacking temporal context, constraining selection before content redundancy can be reliably assessed. Informed compression requires contextual awareness, but acquiring that awareness at full resolution incurs the same quadratic cost compression aims to reduce. State-space models resolve this constraint, as their linear recurrence selectively conditions each token on temporal context at $\mathcal{O}(T)$ cost, producing representations where content redundancy becomes assessable. Building on this, Selective SpatioTemporal Aggregation and Compression (STAC) enriches features via decoupled bidirectional spatial and causal temporal scanning, leveraging recurrence-derived redundancy for hierarchical compression with adaptive thresholds optimised with segmentation objective. STAC achieves 85% token reduction and 1.8$\times$ speedup while surpassing compression-free baselines on reasoning segmentation benchmarks in a zero-shot streaming-compatible setting. ### 1. 论文的核心问题与整体含义（研究动机和背景）
- **研究问题**：视频推理分割要求根据复杂的自然语言查询，在数百帧上进行像素级目标跟踪，导致时空令牌数量巨大（如2分钟视频产生近6.1万令牌），其自注意力的二次计算成本使长视频处理在 MLLM 中几乎不可行。
- **现有方法的局限**：
  - 多数方法采用令牌压缩（帧选择、空间池化、剪枝等），但压缩决策依赖于**缺乏时序上下文的编码器特征**，无法在评估内容冗余前获得充分的上下文感知。
  - 要想获得上下文感知进行明智压缩，又必须处理全分辨率特征，这恰恰是压缩所要避免的二次成本，形成了“先理解再压缩”与“先压缩再理解”之间的矛盾。
- **核心洞察**：状态空间模型（SSM）的线性循环可以在 **O(T)** 成本下为每个令牌注入时序上下文，且已被循环吸收的冗余内容会产生几乎相同的丰富表示，从而为冗余评估提供内在信号。
- **论文目标**：设计一种利用 SSM 循环内在冗余信号的原则性视频令牌压缩方法，实现上下文感知、高效压缩且支持流式推理的视频推理分割。

### 2. 论文提出的方法论：核心思想与关键技术细节
- **整体框架 STAC（Selective Spatiotemporal Aggregation and Compression）**：
  - 将选择性状态空间条件化置于压缩之前，使令牌缩减在融合了时序上下文的特征上进行，而非原始外观特征。
  - 架构分为三阶段：解耦的时空扫描、非对称因果压缩、任务目标联合优化。

- **阶段一：状态引导的时空聚合器（SSA）**
  - **空间聚合**：对每帧进行**双向选择性状态扫描**（前向与后向 Mamba 扫描取均值），克服方向偏差，捕获完整对象边界和帧内语义。
  - **时间聚合**：采用**因果选择性状态扫描**，通过线性循环更新：  
    \(h_{n,t} = \bar{A}_t h_{n,t-1} + \bar{B}_t F_{spatial}[t, n]\)，保留过去上下文并滤除冗余，且支持逐帧增量推理，无需未来帧。
  - 结果：每个令牌获得包含全局时空上下文的丰富特征 \(F_{ST}\)，使内容冗余变得可判别。

- **阶段二：分层状态自适应压缩（HSC）**
  - **自适应时间压缩**：
    - 计算相邻帧归一化相似度 \(s_t\)，通过指数移动平均（EMA）在线估计均值和方差，生成自适应阈值 \(\tau_t = \mu_t + k\sigma_t\)，参数 \(k\) 由轻量 MLP 学习。
    - 连续帧相似度高于阈值则取均值合并，动态实现 60%-70% 帧数缩减。
  - **自适应空间压缩**：
    - 在时间压缩后，计算每个空间位置在时间序列上的跨帧一致性 \(c_n\)，使用位置级别的 EMA 阈值区分静态背景与动态前景。
    - 静态区域的空间令牌被平均合并，前景保留全时分辨率，进一步减少 40%-50% 令牌。
  - 整体实现 ∼85% 令牌缩减，且压缩强度根据内容动态自适应（静态场景压缩率>90%，动作序列仅压缩 10%-50%）。

- **阶段三：任务引导的可微分压缩**
  - 将离散压缩决策（如相似度阈值剪枝）通过直通估计器（Straight‑through estimator）变为可微，使分割损失梯度直接调整压缩策略。
  - 联合优化目标：\( \mathcal{L}_{total} = \mathcal{L}_{seg} + \lambda \mathcal{L}_{comp} \)，\(\mathcal{L}_{comp}\) 引入视频复杂度自适应项，通过 \(1/\sqrt{T_i}\) 归一化防止长视频过度压缩。
  - 最终形成闭环：解耦扫描提供上下文 → HSC 据此压缩 → 分割目标反传校准压缩策略。

### 6. 论文的主要结论与发现
- STAC 可在 **85% 令牌压缩** 和 **1.8倍加速** 下，在指代分割（Ref‑DAVIS17: 73.6 J&F）和零样本推理分割（ReasonVOS: 52.3 J&F）上超越多数无压缩方法。

- 因果时间扫描设计使模型支持逐帧在线处理，在推理时保持恒定延迟（约200ms/帧），而双向方法无法做到。
- 适应性分层压缩（先时间后空间）是关键：空间优先会破坏运动信号，时间优先能保留关键动态信息。
