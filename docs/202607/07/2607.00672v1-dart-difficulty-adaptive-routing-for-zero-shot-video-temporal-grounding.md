## 摘要
零样本视频时序定位（VTG）在没有任务特定训练的情况下，根据自然语言查询在未剪辑视频中定位事件。现有方法依赖帧-查询特征匹配，这对简单事件足够，但难以处理需要理解时间顺序和因果结构的复杂多阶段查询——我们称之为推理差距。我们提出了DART（面向时序定位的难度自适应路由），通过将难度感知路由与大型视觉-语言模型中的结构化推理相结合来弥合这一差距。一个以查询为条件的行列式点过程（DPP）发挥双重作用：选择多样化、与查询相关的关键帧作为时间证据，并提供谱熵作为难度指标。简单查询被路由到快速路径进行直接预测，而复杂查询则遵循慢速路径，采用时间标记提示，将定位分解为全局事件分析、逐帧时间角色注释和边界提取。在Charades-STA和ActivityNet Captions上，DART在相同分布和多种分布外设置下均实现了最先进的零样本性能，与最强基线相比mIoU提高了多达3.5个百分点，同时使用的帧数减少了超过7倍。项目主页位于https://dart-vtg.github.io/。

## Abstract
Zero-shot video temporal grounding (VTG) localizes events in untrimmed videos from natural language queries without task-specific training. Existing methods rely on frame-query feature matching, which suffices for simple events but struggles with complex multi-stage queries that require understanding temporal ordering and causal structure -- a disparity we call the reasoning gap. We propose DART (Difficulty-Adaptive Routing for Temporal Grounding), which bridges this gap by coupling difficulty-aware routing with structured reasoning in large vision-language models. A query-conditioned Determinantal Point Process (DPP) serves a dual role: selecting diverse, query-relevant keyframes as temporal evidence, and providing spectral entropy as a difficulty indicator. Simple queries are routed to a Fast path for direct prediction, while complex queries follow a Slow path with Temporal Markup Prompting, which decomposes localization into global event analysis, per-frame temporal role annotation, and boundary extraction. On Charades-STA and ActivityNet Captions, DART achieves state-of-the-art zero-shot performance across both identically distributed and multiple out-of-distribution settings, improving mIoU by up to 3.5 points over the strongest baseline while using over 7 times fewer frames. The project homepage is available at https://dart-vtg.github.io/.

### 1. 论文的核心问题与整体含义（研究动机和背景）
- 零样本视频时序定位（Zero-shot VTG）旨在没有任务特定训练数据的情况下，根据自然语言查询在未剪辑视频中定位目标事件的起止时间。
- 现有方法主要依赖帧级特征匹配（frame-query feature matching），对于简单查询（如“人坐下”）效果尚可，但对于需要理解时间顺序、因果关系、条件逻辑的复杂多阶段查询（如“他在反转后下马并落在垫子上”），性能骤降。
- 作者将这一系统性的性能差距定义为**推理差距（reasoning gap）**，即特征匹配无法捕捉跨帧的时间依赖与事件组成结构。
- 因此，该工作致力于弥补零样本 VTG 中的推理差距，使模型能够自适应地决定何时需要结构化推理，而不是对所有查询都采用固定的处理流程。

### 2. 论文提出的方法论
- **整体框架：DART（Difficulty-Adaptive Routing for Temporal Grounding）**  
  核心思想是将难度感知路由与大型视觉-语言模型（LVLM）的结构化推理相结合，通过查询条件行列式点过程（DPP）同时完成关键帧筛选和难度判别，再路由至快速/慢速双路径进行定位。
- **关键技术细节**
  - **查询条件的DPP关键帧选择**  
    构造查询条件核矩阵 \(\tilde{L}_{ij} = \sqrt{w_i} L_{ij} \sqrt{w_j}\)，其中 \(L_{ij}\) 为帧间视觉-时间相似度（视觉高斯核×时间高斯核），\(w_i\) 为帧与查询的 sigmoid 相似度。通过贪心最大化行列式质量函数，按边际增益 \(\Delta(e|S) = \log \det(L_{S\cup\{e\}}) - \log \det(L_S) + \log w_e\) 选取关键帧，同时满足**查询相关性**与**帧间多样性**。自适应停止准则：当相对增益 \(\Delta_k/\Delta_1 < \varepsilon\) 时终止。
  - **基于谱熵的难度路由**  
    对 \(\tilde{L}\) 做特征分解，将特征值归一化为概率分布 \(q_i = \lambda_i/\sum_j \lambda_j\)，计算谱熵 \(H_{\text{spectral}} = -\frac{1}{\log M} \sum_{i=1}^M q_i \log q_i\)。低熵表示事件阶段少（简单查询），高熵表示多阶段结构（复杂查询）。以阈值 \(\theta\) 路由：\(H_{\text{spectral}} \le \theta\) → 快速路径；否则 → 慢速路径。
  - **双路径时间定位**
    - **快速路径（Fast Path）**：直接将所选关键帧与时间戳送入 LVLM，令其输出起止时间戳，无需中间推理。
    - **慢速路径（Slow Path）**：采用**时间标记提示（Temporal Markup Prompting, TMP）**，分三步：  
      ① 全局时间推理：拆解查询的关键动作及时间关系，勾画事件大致范围。  
      ② 逐帧时间角色标注：为每一关键帧分配 {before, start, during, end, after} 中的单调非递减角色。  
      ③ 边界提取：取第一个“start”帧和最后一个“end”帧作为时间边界。

### 6. 论文的主要结论与发现
- 零样本 VTG 中存在显著的推理差距：复杂查询比简单查询的 mIoU 低约 11.6 个百分点，现有特征匹配方法无法弥补。
- DART 通过查询条件 DPP 选择多样化关键帧，其谱熵能够可靠地指示查询复杂度，并自适应地路由到快速或慢速推理路径。
- 在 Charades-STA 和 ActivityNet Captions 上，DART 在全部 IID、OOD 及跨数据集设置下均取得最先进零样本性能，mIoU 最高提升 3.5 点（TAG），同时使用的帧数减少 7 倍以上。
- 自适应路由带来的效率提升显著：相比对所有查询强制推理的慢速路径，自适应路由在降低 23.5% 推理时间的同时还提升了 mIoU。
- 关键帧质量和结构化推理对于性能至关重要；缺乏高质量的帧选择会削弱路由和推理的收益。

### 7. 优点
- **新颖的难度自适应范式**：首次在零样本 VTG 中引入基于谱熵的难度判断和自适应路由，决定“何时推理”与“如何推理”同等重要。
- **统一的框架**：查询条件 DPP 同时解决关键帧选择和难度测量，一个谱分解即完成两个任务，设计精巧。
- **结构化提示 TMP**：与通用思维链相比，TMP 为时序定位定制了三步规整的推理步骤，强制输出角色标签和时序一致性，明显优于无约束推理。
- **综合实验验证**：覆盖 IID、多种 OOD 及跨域设置，对比面广，消融实验充分，且提供了效率分析，证明方法高效实用。
- **性能与效率双优**：在多个基准上超越所有零样本方法，甚至匹敌或超越弱监督方法，同时大幅度减少帧输入和推理时间。

### 8. 不足与局限
- **对 LVLM 的依赖**：方法性能受限于所使用的 LLaVA-1.6-7B 的视觉-语言能力，更换更强或更弱的 LVLM 可能会有波动，论文未对多种 LVLM 进行敏感性分析。
- **路由阈值 θ 和停止阈值 ε 等超参数** 需在验证集上调整，不同数据集间可能需要重新设定，自适应程度有限。
- **慢速路径依赖 LVLM 的生成质量**：TMP 要求模型产生格式严格的时间标签和推理解释，可能受模型幻觉影响，论文未深入分析失败案例类型。
- **应用场景**：目前仅针对单查询、单段定位，未涉及多事件、长视频或流式视频的扩展。
- **零样本仍不算真正零资源**：使用了预训练的 LVLM（需要大量计算和数据），但这是当前零样本 VTG 的普遍前提，并非 DART 特有局限。
