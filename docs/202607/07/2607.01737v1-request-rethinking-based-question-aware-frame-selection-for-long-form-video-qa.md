## 摘要
近年来，多模态大语言模型（MLLMs）极大地推动了视频理解的发展，然而在固定输入令牌预算下，长视频问答仍具挑战性，其中均匀采样在证据定位方面效率低下。我们提出ReQuest，一种由不确定性驱动、问题自适应关键帧选择流程，通过选择性计算将问题意图与相关视频内容对齐。ReQuest集成了（i）一个从MLLM生成的监督中蒸馏出的轻量级问题感知选择器，（ii）再思考路由，仅当模型不确定时根据长度自适应准则触发额外推理，以及（iii）不确定性引导的自适应非极大值抑制，在根据问题难度调整间隔的同时选择时间上多样化的帧。作为一种即插即用的方法，ReQuest在不修改或微调底层MLLM的情况下提升了长视频问答性能。在Video-MME、MLVU和LongVideoBench上的实验表明，在具有竞争力的计算成本下准确率持续提高，尤其在中等和长视频场景中改进显著。

## Abstract
Recent multimodal large language models (MLLMs) have substantially advanced video understanding, yet long-form video QA remains challenging under fixed input token budgets, where uniform sampling can be inefficient for evidence localization. We propose ReQuest , an uncertainty-driven, question-adaptive keyframe selection pipeline that aligns question intent with relevant video content through selective computation. ReQuest integrates (i) a lightweight question-aware selector distilled from MLLM-generated supervision, (ii) Re-thinking Routing that triggers additional inference only when the model is uncertain with a length-adaptive criterion, and (iii) uncertainty-guided adaptive non-maximum suppression that selects temporally diverse frames while adjusting spacing based on question difficulty. As a plug-andplay method, ReQuest improves long-video QA without modifying or fine-tuning the underlying MLLM. Experiments on Video-MME, MLVU, and LongVideoBench demonstrate consistent accuracy gains with competitive computational cost, with particularly strong improvements in medium and long video regimes.

- **核心问题**：多模态大语言模型（MLLMs）在回答长视频问题时，因输入令牌（token）数量固定，只能选取部分视频帧（如均匀采样），容易遗漏关键证据。视频越长，“大海捞针”问题就越严重。
- **研究动机**：
    - 均匀采样简单但低效，因冗余帧挤占关键帧空间。
    - 基于帧-问题相似度检索虽有效，但若问题中不直接包含答案相关物体关键词（如“他休息那天用了什么工具？”答案不在帧中），相似度无法捕捉深层语义。
    - 基于MLLM本身进行帧选择虽能深层理解语义，但计算开销大，且其多阶段采样管线以稀疏均匀采样开始，同样有信息损失风险。
- **整体目标**：设计一种低成本、问题自适应、能全局探索视频的关键帧选择方法，在不修改底层MLLM的前提下，提升长视频问答准确率，特别是在证据稀疏的中、长视频上。

### 2. 论文提出的方法论：核心思想、关键技术细节、算法流程
ReQuest 的流程分为三阶段，核心思路是以MLLM的响应不确定性为引导，自适应选择关键帧。

- **第一阶段：问题感知的关键帧选择器**
    - **核心思想**：训练一个轻量级选择器，模拟MLLM的问题理解能力，评估每个视频段（或帧）对回答问题的“贡献度”。
    - **伪标签生成**：用MLLM分别对含视频段、全部遮罩（仅保留文本）两种输入的回答正确选项的概率，相减得到每个视频段纯视觉贡献分数 $\Delta c_i$。
    - **选择器架构**：利用BLIP进行视觉-问题交叉注意力融合（而非双编码器独立编码），再将融合特征送入一个轻量级Transformer，输出贡献预测分数，用Smooth-ℓ1损失回归到伪标签 $\Delta c_i$。
- **第二阶段：不确定性驱动的再思考路由**
    - **核心思想**：不所有问题都重选帧，而是根据模型初次预测的置信度动态决定。模型不确定 → 需要更多证据 => 进入“再思考”流程。
    - **路由算法**：先用均匀采样的帧让MLLM回答，计算选项概率熵作为不确定性 $u$。结合视频长度 $N$ 动态调整阈值 $\tau_{\text{eff}} = \tau_0 - \gamma_{\text{len}} \cdot \min(1, \frac{N}{N_{\max}})$（视频越长，阈值越低，越易触发再思考）。熵高于阈值就触发重选择，否则直接输出答案。
- **第三阶段：自适应非极大值抑制（NMS）帧采样**
    - **核心思想**：在获得选择器对每帧的分数后，用贪婪NMS依次选取最高分帧，并抑制时序邻近帧。抑制窗口大小由不确定性动态调节。
    - **自适应机制**：不确定性 $u$ 越高 → 有效NMS间隔 $\delta$ 缩小，允许更密集地在局部区域采样（类似“放大镜”效果）。$\delta$ 从 $u$ 经幂函数映射，并在 $s_{\min}$ 到 1 区间缩放基长度 $b_s$。

### 6. 论文的主要结论与发现
- ReQuest 以即插即用的方式，在不微调任何MLLM的情况下，在多个长视频问答基准上取得了稳定而显著的提升，尤其在证据稀疏的中、长视频场景提升最大。
- 方法通过不确定性引导的自适应重思考，相比单纯均匀采样或固定窗口选择，可以更准确且计算高效地定位对回答问题最关键的稀少证据帧。

- 即使是高帧数承载的MLLM（如Qwen3-VL），盲目增加帧数未必带来增益；相反，用更少的、经问题自适应筛选的帧反而能获得更好的准确率，同时规避过度输入的冗余。

### 7. 优点
- **即插即用、架构通用**：无需修改基座MLLM，可轻易迁移至不同模型（LLaVA-OneVision、Qwen2.5/3-VL等）。
- **精巧的自适应设计**：路由与NMS区间的动态调整，是“将模型响应反馈给输入选择”这一思想的优雅实现，实现了精度与效率的平衡。
- **监督信号创新**：利用MLLM回答概率差构建“贡献度”伪标签，比单纯用图文相似度更能反映语义线索的真实有效性，训练信号价值更高。
- **实验设计严谨**：多维度对比、消融充分，特别关注了随机种子、二次推理等潜在干扰因素，确保增益来自方法本身。

### 8. 不足与局限
- **三段式管线，有累计延迟**：方法仍依赖先密集解码视频帧，再提取特征筛选，处理起步延迟较高（如1fps下总延迟超100秒），实时性有限。
- **依赖初始均匀采样质量**：路由阶段的“初次回答”若质量太差导致不确定性异常，可能误导是否触发的决策。超参数（路由阈值）需根据任务手动调整，方法有轻微手动调参依赖。
- **实验限制在选择题为主**：虽然开放域问答也有验证，但核心评测、伪标签产生均依赖多选选项的对数几率。在完全开放式问答上，不确定性度量、贡献度计算的有效性可能降低或需重新设计。
- **文本偏差风险**：虽然通过去偏基线校正，但贡献度$\Delta c_i$ 训练选择器时，仍可能受原 MLLM 自身语言先验和幻觉影响，继承其偏差。
- **算力需求**：伪标签生成需64块GPU耗时12小时，对复现构成较大开销。
