## 摘要
对话语音合成（CSS）旨在用户与智能体的交互中合成具有类人情感表达和上下文一致性的语音。现有的CSS方法由于有限的预定义情感标签空间（例如七类情感）而难以渲染真实的人类情感，同时多轮对话历史中的冗余多模态令牌干扰了上下文理解。为了解决这些问题，我们提出了AuEmoChat，一个用于真实情感理解与渲染的CSS框架。首先，我们开发了AuEmoCodec，它通过有限标量量化从大规模情感语音中学习离散的真实情感令牌空间，从而能够比有限的基本情感类别实现更真实的情感表示。我们进一步提出了AuEmoToMe，一种以真实情感为指导的令牌合并算法，该算法在保留情感相关上下文的同时合并多模态对话历史中的冗余令牌。我们将其集成到一个自回归文本到语音模型中，以预测目标真实情感令牌和语音令牌。最后，我们提出了真实情感流匹配，通过联合条件作用于合并后的对话上下文、目标真实情感和声学先验来渲染语音。在NCSSD-EmCap数据集上的大量实验表明，AuEmoChat优于最先进的CSS基线，并生成更具表现力和真实感的情感语音。

## Abstract
Conversational Speech Synthesis (CSS) aims to synthesize speech with human-like emotional expression and contextual consistency in user-agent interactions. Existing CSS methods struggle to render authentic human emotions due to limited predefined emotion label spaces (e.g., seven emotion categories), while redundant multimodal tokens in multi-turn dialogue history interfere with context understanding. To address these issues, we propose AuEmoChat, a CSS framework for authentic emotion understanding and rendering. First, we develop AuEmoCodec, which learns a discrete authentic emotion token space from large-scale emotional speech via finite scalar quantization, enabling a more authentic emotion representation than limited basic emotion categories. We further propose AuEmoToMe, an authentic-emotion-guided token merging algorithm that merges redundant tokens in multimodal dialogue history while preserving emotion-relevant context. We integrate it into an autoregressive text-speech model to predict the target authentic emotion token and speech tokens. Finally, we propose Authentic Emotion Flow Matching, which renders speech by jointly conditioning on merged dialogue context, target authentic emotion, and acoustic priors. Extensive experiments on the NCSSD-EmCap dataset demonstrate that AuEmoChat outperforms state-of-the-art CSS baselines and generates more expressive and authentic emotional speech.

  - **多模态上下文令牌冗余**：多轮对话历史中文本和语音令牌序列越来越长，包含大量与情感无关的冗余信息，严重干扰情感理解与高质量语音生成。
- **整体含义**：AuEmoChat 旨在突破上述两大瓶颈，构建一个能够理解并渲染真实、细腻情感的 CSS 框架。

## 2. 论文提出的方法论
### 2.1 核心思想
- 不再使用离散情感标签，而是学习一个能够表达丰富、细腻情感状态的**离散真实情感令牌空间**。
- 在对话上下文建模中，通过**情感引导的令牌合并**减少冗余，保留情感关键信息。
- 在语音生成阶段，采用**真实情感流匹配**，联合建模上下文、目标情感和声学先验，生成情感逼真、上下文一致的语音。

### 2.2 AuEmoCodec：真实情感令牌空间学习
- **目标**：从大规模情感语音中通过有限标量量化（FSQ）学习离散的情感令牌，使每一个令牌代表一种细腻的真实情感状态。
- **实现**：
  - 使用情感编码器提取语音的情感表示，再经 FSQ-Down 投影并量化（量化等级 [8,5,5,5]），得到 Token `f_E`。
  - 利用 FSQ-Up 将量化码重建为 7 个基本情绪轴上的感知情感得分 `P_ae`。
  - 采用 Gemini-2.5-Flash 作为评判模型，为语音样本标注 7 轴情感感知分值 `GT_ae`，训练目标为最小化均方误差（MSE），促使令牌捕捉真实情感的微妙差异。
- **结果**：构建由 1000 个码本（激活约 750 个）组成的 AuEmo 令牌空间，每个令牌对应一个真实情感状态。

### 2.3 AuEmoToMe：情感引导的多模态令牌合并
- **问题**：多模态对话历史中文本和语音令牌数随轮次快速膨胀，冗余量大，影响目标情感和语音令牌的预测。
- **方法**：
  - 对每一句历史话语，分别将其文本令牌和语音令牌划分为源集合和目的集合（基于合并比例 ρ）。
  - 计算源与目的令牌间的余弦相似度，并选择最相似的 top‑k 对进行合并。
  - **情感引导加权**：将当前话语的 AuEmo 令牌作为情感锚点，分别计算待合并两个令牌与情感锚点的相似度，归一化后作为融合权重 `w_a` 和 `w_b`，加权聚合到目的令牌上。这样在压缩冗余的同时，保留了与情感相关的关键信息。
- **作用**：压缩后的历史令牌 `f_his_merged` 更紧凑且富含情感线索，用于后续自回归模型预测目标 AuEmo 令牌和语音令牌。

### 2.4 自回归目标预测
- 将合并后的历史令牌与目标话语的说话人令牌、文本令牌拼接，输入自回归语言模型。
- 模型首先预测目标话语的**真实情感令牌** `f_E_N`，然后在此基础上生成**语音令牌序列**（以终止符结束）。
- 公式化地表示为条件概率建模，将情感理解分离为显式步骤。

### 2.5 真实情感流匹配（Authentic Emotion Flow Matching）
- **输入条件**：
  - 合并上下文编码器：对 `f_his_merged` 双向 GRU 建模，得到上下文条件 `F_ctx`。
  - AuEmo 编码器：将预测的 `f_E_N` 投影为情感条件 `F_e`。
  - 语音令牌序列转换的声学先验 μ。
- **流匹配生成**：
  - 定义高斯噪声到目标梅尔谱图的插值路径，训练一个网络预测向量场，使噪声沿路径向目标谱图移动。
  - 损失函数为预测向量场与真实运输方向之间的 L2 距离 `L_fm`。
- **情感分类引导（ACG）**：
  - 引入预训练的 AuEmo 分类器，对中间梅尔状态计算属于目标情感令牌的概率梯度，仅在运输后期（t > 0.7）施加引导，使生成轨迹更贴合目标真实情感。
- **推理**：从噪声开始，通过解常微分方程生成梅尔谱图，再经 HiFi-GAN 转换为波形。
