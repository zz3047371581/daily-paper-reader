## 摘要
第11届野外情感行为分析（ABAW11）多任务学习挑战赛要求基于官方s-Aff-Wild2图像，构建一个统一的系统来预测效价-唤醒度、类别表情和面部动作单元。尽管这些任务通过面部行为自然关联，但我们的验证实验表明，它们受益于不同的视觉特征、时序处理策略、融合机制和校准程序。本文研究了面向ABAW11多任务情感行为分析的任务自适应特征融合。我们首先在外部的面向表情的面部图像集上微调两个预训练视觉骨干网络DINOv2 ViT-L和DINOv3 ConvNeXt-base，然后冻结它们以从官方ABAW11数据中提取互补的帧级特征。在这些冻结特征之上，我们系统地比较了帧级预测头、时序卷积头、事后时序平滑、LightGBM模型、特征拼接、门控融合、残差融合、后期logit融合、阈值校准和共享多任务学习结构。最终系统选择了任务特定的融合和预测策略，而不是强制所有任务共享单一架构。在ABAW11验证集上，所选系统实现了0.4222的EXPR宏观F1、0.5402的AU宏观F1和0.6717的平均VA CCC，总体验证得分为1.6341。结果表明，冻结视觉特征的任务自适应融合是ABAW风格多任务情感行为分析的一种简单有效的策略。

## Abstract
The 11th Affective Behavior Analysis in-the-wild (ABAW11) Multi-Task Learning Challenge requires a unified system to predict valence-arousal, categorical expressions, and facial action units from the official s-Aff-Wild2 images. Although these tasks are naturally related through facial behavior, our validation experiments show that they benefit from different visual features, temporal processing strategies, fusion mechanisms, and calibration procedures. In this paper, we study task-adaptive feature fusion for ABAW11 multi-task affective behavior analysis. We first adapt two pretrained visual backbones, DINOv2 ViT-L and DINOv3 ConvNeXt-base, on an external expression-oriented facial image set and then freeze them to extract complementary frame-level features from the official ABAW11 data. On top of these frozen features, we systematically compare frame-level prediction heads, temporal convolutional heads, post-hoc temporal smoothing, LightGBM models, feature concatenation, gated fusion, residual fusion, late logit fusion, threshold calibration, and shared MTL structures. The final system selects task-specific fusion and prediction strategies rather than forcing all tasks to share a single architecture. On the ABAW11 validation set, the selected system achieves an EXPR macro-F1 of 0.4222, an AU macro-F1 of 0.5402, and a mean VA CCC of 0.6717, resulting in an overall validation score of 1.6341. The results suggest that task-adaptive fusion of frozen visual features is a simple and effective strategy for ABAW-style multi-task affective behavior analysis.

  因此，本文的核心问题是：**能否为每个任务选择各自最合适的特征融合与预测策略，而非使用统一的多任务共享架构？** 论文的整体含义在于验证“任务自适应特征融合”在多任务情感分析中的有效性与简单性。

## 2. 方法论

### 2.1 核心思想  
为三个任务分别建立**解耦的专家路线**，而不是端到端训练统一的共享网络。具体做法分两步：  
1. 利用外部表情数据适配两个预训练骨干网络（DINOv2 ViT‑L 与 DINOv3 ConvNeXt‑base），使它们获得更强的面部表征能力，随后冻结。  
2. 在冻结的骨干特征之上，为每个任务从**特征源、融合方式、时序处理、预测头、校准策略**等多个维度中选出验证集上最优的组合。

### 2.2 关键技术与流程  
- **冻结视觉特征提取**  
  对每个输入帧 \(x_t\)，提取两个1024维特征向量：  
  \[
  \mathbf{z}^{vit}_t = f_{\text{vit}}(x_t), \quad \mathbf{z}^{cnn}_t = f_{\text{cnn}}(x_t)
  \]  
  其中 \(f_{\text{vit}}\) 是适配后的DINOv2 ViT‑L（保留 class‑token），\(f_{\text{cnn}}\) 是适配后的DINOv3 ConvNeXt‑base（取全局池化后表示）。两个骨干网络在外部表情集上微调后丢弃分类头，用于离线提取全部ABAW11帧特征，下游实验共享同一组特征。

- **外部骨干适配**  
  - ViT‑L：冻结前8个Transformer块和patch embedding，剩余层配合一个四专家门控混合‑专家分类器进行表情分类训练。  
  - ConvNeXt‑base：整个骨干与线性分类器联合微调。  
  适配数据来自AffectNet与RAF‑DB重组的8类表情集，不包含任何Aff‑Wild2图像。

- **下游专家池**  
  冻结特征之上，系统比较了多种配置：  
  - **帧级预测头**：线性、MLP、BigMLP，根据任务分别输出（VA：tanh‑限幅回归；EXPR：softmax分类；AU：sigmoid二分类）。  
  - **时序处理**：  
    · 时序卷积头（TCN）：将特征投影后经过多个残差扩张卷积块，在视频窗口内建模。  
    · 事后平滑：对帧级预测在视频内做均值或中值平滑（不跨视频），平滑窗口大小可调。  
  - **非神经基线**：用LightGBM在ViT‑L特征上训练，并利用局部时序均值与偏差进行简单上下文增强。  
  - **特征融合（双特征）**：  
    · 直接拼接  
    · 门控融合：对两个归一化投影后的特征计算软注意力门控，再用残差门控修正ViT‑L主支。  
    · 残差融合：将ConvNeXt‑base特征经门控后直接加到ViT‑L支路（残差门偏置初始化‑4）。  
  - **后期logit/log‑prediction融合**：对两个独立训练的专家输出取加权和 \(\hat{p}_t = (1-\alpha)\hat{p}^{vit}_t + \alpha\hat{p}^{cnn}_t\)，\(\alpha\) 在验证集选取。  
  - **阈值校准**：AU任务在验证集上遍历阈值（0.01‑0.99）使每个AU的F1最大化。  
  - **共享MTL基线**：包括共享MLP、任务‑令牌Transformer、残差关系模块等，用于对比统一架构的效果。

- **训练目标**  
  - VA：MSE + 一致相关系数损失（λ_mse=0.5，λ_ccc=1）。  
  - EXPR：类加权交叉熵 + 标签平滑（权重基于类样本数的平方根归一化）。  
  - AU：正样本加权二元交叉熵。  
  共享MTL的联合损失为三项损失直接求和。

### 2.3 最终任务自适应选择  
通过验证集实验，为三个任务分别确定的配置：  
- EXPR：ViT‑L与ConvNeXt特征拼接 + 中值平滑。  
- AU：ViT‑L与ConvNeXt独立训练的logit后融合 + 阈值校准。  
- VA：残差特征融合 + 平滑（但五折交叉验证发现ViT‑L时序卷积更稳定，最终系统采纳了时序卷积）。

## 3. 实验设计

- **数据集与基准**  
  - 训练与验证：ABAW11官方s‑Aff‑Wild2数据集，仅使用标注有效的帧，缺失标签的样本在对应任务中忽略。  
  - 外部适配数据：从AffectNet和RAF‑DB组建的8类表情集合，不含Aff‑Wild2图像。

- **对比的方法范围**  
  - **帧级头消融**（线性、MLP、BigMLP），分别针对ViT‑L和ConvNeXt特征。  
  - **单特征专家比较**：ViT‑L、ConvNeXt、LightGBM（使用ViT‑L特征），并搭配帧级预测、平滑、时序卷积三种处理。

  - **共享MTL结构**：共享MLP、任务‑令牌Transformer、残差关系模块等，使用ViT‑L、ConvNeXt或拼接特征，不引入显式时序建模。

## 6. 主要结论与发现

- 三个任务的最优配置**彼此不同**：EXPR偏好特征拼接+平滑；AU偏好后期logit融合+阈值校准；VA在公开验证集上以残差融合+平滑得分最高，但在视频级交叉验证中时序卷积更稳定。  
- 任务自适应融合的总验证得分达到**1.6341**，远高于最优化共享MTL基线（1.3598），表明为每个任务定制策略的必要性。  
- 冻结视觉特征加轻量下游专家，是一种**简单有效**且易于复现的多任务情感分析方案。  
- LightGBM等树模型在AU任务上竞争力较强，说明非神经网络模型在该任务上仍可有效利用冻结表征。  
- 通过外部表情数据的针对性适配，骨干网络无需接触ABAW11标注即可获得有益的面部先验。
