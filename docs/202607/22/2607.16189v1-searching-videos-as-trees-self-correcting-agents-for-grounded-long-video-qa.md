## 摘要
有依据的长视频问答要求回答关于长视频的问题，同时定位支持答案的短证据区间。近期的智能体方法将此任务构建为使用单一crop_video(start, end)动作的多轮探索，它支持从粗到细的收缩，但不提供从细到粗回溯的原语。因此，这些智能体通常会过早收敛，无法从早期错误中恢复。我们提出VideoTreeSearch (VTS)，一个将依据性长视频问答视为在自适应时间树上进行迭代自校正搜索的框架。VTS根据视觉场景边界构建非均匀树，使得每个节点对应一个语义连贯的片段，并训练智能体通过四个离散操作在树上导航：zoom_in、zoom_out、shift和answer。这些操作将回溯和恢复作为显式的、可学习的原语而非隐式行为暴露出来。为了训练这种导航，我们引入了一个轨迹合成流程，生成通过树的多步路径，包括故意误入错误分支随后恢复。我们使用这些轨迹进行监督微调，随后使用依据性和答案准确性奖励进行强化学习。在三个有依据长视频问答基准（CG-Bench、Haystack-LVBench、Haystack-Ego4D）上，VTS在CG-Bench上比最强先前智能体方法提高了+12.5 mIoU，在Haystack-Ego4D上提高了+7.4 T-F1。学到的策略还迁移到一般长视频问答，在Video-MME、MLVU和LVBench上超越了所有先前智能体基线，最高提升+7.1个准确度点。消融实验证实，自校正分层搜索是这些收益背后的核心机制：移除自适应下降或显式回溯都会显著降低性能。代码可在https://github.com/CeeZh/VTS获取。

## Abstract
Grounded long-video question answering (Grounded LVQA) requires answering a question about a long video while localizing the short evidence interval that supports the answer. Recent agentic methods frame this task as multi-turn exploration with a single crop_video(start, end) action, which supports coarse-to-fine narrowing but provides no primitive for fine-to-coarse backtracking. As a result, these agents typically converge prematurely and cannot recover from an early mistake. We propose VideoTreeSearch (VTS), a framework that casts grounded LVQA as iterative self-correcting search over an adaptive temporal tree. VTS constructs a non-uniform tree from visual scene boundaries so that each node corresponds to a semantically coherent segment, and trains an agent to navigate the tree through four discrete operations: zoom_in, zoom_out, shift, and answer. These operations expose backtracking and recovery as explicit, learnable primitives rather than implicit behaviors. To train this navigation, we introduce a trajectory synthesis pipeline that produces multi-step paths through the tree, including deliberate detours into incorrect branches followed by recovery. We use these trajectories for supervised fine-tuning, followed by reinforcement learning with grounding and answer-accuracy rewards. On three Grounded LVQA benchmarks (CG-Bench, Haystack-LVBench, Haystack-Ego4D), VTS outperforms the strongest prior agentic methods by +12.5 mIoU on CG-Bench and +7.4 T-F1 on Haystack-Ego4D. The learned policy also transfers to general long-video QA, surpassing all prior agentic baselines on Video-MME, MLVU, and LVBench by up to +7.1 accuracy points. Ablations confirm that self-correcting hierarchical search is the central mechanism behind these gains: removing either adaptive descent or explicit backtracking substantially degrades performance. Code is available at https://github.com/CeeZh/VTS.

### 1. 论文的核心问题与整体含义

- **研究背景**：有依据的长视频问答（Grounded LVQA）要求模型不仅回答关于长视频的问题，还要同时定位出支持该答案的短证据时间区间。传统的均匀帧采样难以捕捉稀疏的关键证据，而现有智能体方法通常依靠单一的 `crop_video(start, end)` 动作进行多轮探索。
- **存在的局限**：这类连续裁剪动作虽然能从粗到细地收缩，但缺乏从细到粗的“回溯”原语。一旦智能体在早期选择错误的视频区域，就会过早收敛、无法恢复，导致搜索过程不对称且缺乏结构化的纠错能力。
- **整体含义**：论文提出 **VideoTreeSearch (VTS)** 框架，将长视频组织为一棵自适应时间树，并将搜索过程建模为在树上的迭代自校正导航。通过引入离散的缩放、倒退、平移动和回答操作，使“回溯与纠错”成为显式可学习的原语，从而系统性地解决现有方法易陷入错误且无法恢复的问题。

### 2. 论文提出的方法论

- **核心思想**：将长视频递归分割为一棵非均匀的语义树，每个树节点对应一个语义连贯的视频片段；智能体通过在树上的离散动作序列实现多尺度分层搜索与自校正。
- **自适应时间树构建**：
  - 以整个视频为根节点，利用 CLIP 提取帧嵌入，计算相邻帧余弦距离作为视觉变化。
  - 当距离超过自适应阈值 τ = mean(δ) + k·std(δ) 时设置分割边界。
  - 每层节点子节点数限定在 3~8 个，叶子节点时长小于 64 秒。树按需惰性构建，仅当智能体访问时展开。
- **树基动作空间**：
  - **zoom_in(c)**：下降到当前节点的第 c 个子节点。
  - **zoom_out()**：上升到父节点。
  - **shift(s)**：平移到同一个父节点下的第 s 个兄弟节点。
  - **answer(ŷ, [ts, te])**：终止搜索，输出最终答案和证据区间。
  这些操作将“向下探索”与“向上/平移纠正”完全解耦，使回溯成为显式的学习信号。
- **多轮导航与记忆**：
  - 每轮智能体接收当前节点及其子节点的帧画面，结合紧凑记忆（已展开的树结构、各节点描述、历史动作记录）生成推理轨迹并选择动作。
  - 非当前节点仅用简短文本描述表示，以控制上下文长度。
- **轨迹合成**：为训练纠错能力，合成包含“故意误入错误分支然后恢复”的完整导航路径。控制器在正确状态下选择与问题最相关的子节点；进入错误分支后随机决定继续深入或执行逆操作恢复，但强制在错误分支停留两轮后必须恢复。每个动作配合理由推理文本，由 DeepSeek‑R1 生成。
- **训练流程**：
  - **监督微调 (SFT)**：仅对期望模仿的正确动作（如正确的 zoom_in、从错误节点 zoom_out、移动到正确兄弟节点等）计算损失，故意犯错的动作被遮盖。
  - **强化学习 (RL)**：在 SFT 基础上使用 GRPO 进一步优化，奖励函数包含格式正确性、时间 IoU（预测证据区间与真值重叠度）和答案准确性三部分，并加入 KL 惩罚防止策略漂移。
- **实现细节**：基础模型为 Qwen3-VL-8B，k=1.5；每轮采样最多 64 帧（1 fps）。训练数据来自 CG-Bench、Haystack-Ego4D 和自动构建的 LongClueQA 语料库，共 6537 条 QA 对。

### 6. 论文的主要结论与发现

- VTS 在所有三个依据性长视频 QA 基准上大幅超越先前最优智能体方法，尤其在最长视频基准 Haystack-Ego4D 上提升最大（+7.4 T-F1），且在 CG-Bench 上 mIoU 提升 +12.5。

- 自校正分层搜索是核心机制：移除自适应层次降采样或显式回溯操作（zoom_out/shift）均导致性能显著下降。
- 合成包含纠错路径的训练轨迹对于让智能体学会有效搜索至关重要，仅用最优路径训练效果很差。
- 树基离散动作比连续裁剪动作更能支持多轮探索与自我纠正（VTS 平均 4.8 轮，约 60% 轨迹触发回溯）。

### 7. 优点（方法或实验设计的亮点）

- **创新性动作空间**：将回溯和恢复定义为独立、显式的离散动作，使智能体可以学习何时以及如何放弃错误路径，解决了现有连续裁剪动作的非对称性缺陷。
- **结构化、多尺度的搜索空间**：基于视觉场景边界构建的非均匀树，使定位降维为选节点，无需从原始时间戳回归，更加稳定且语义对齐。
- **针对性轨迹合成**：设计了一套包含“故意犯错-恢复”路径的轨迹生成方式，为学习纠错策略提供了关键监督信号。
- **两阶段训练策略**：SFT 只对期望动作监督，RL 进一步通过结果奖励优化，有效平衡模仿学习与探索。
- **实验严谨性**：大量消融实验和相同骨干对比，清晰地揭示了各个组件的贡献；同时展示了策略跨任务的迁移能力。

### 8. 不足与局限

- **问答范围限定**：目前动作空间仅生成单个证据区间，无法处理证据分布在多个不连续时间段的复杂问题。
- **树构建依赖视觉质量**：自适应分割依赖 CLIP 提取的场景边界，在视觉内容变化不明显或高度同质的长视频中可能失效，导致树结构不够精确。
- **训练数据依赖外部模型**：轨迹合成中用到的问句生成、得分及推理文本均由其他 VLM/LLM（如 Qwen3-VL、DeepSeek-R1）产生，训练信号质量受这些模型能力限制。
- **问答形式单一**：当前仅考察多选问题的准确率与定位精度，未涉及开放式回答生成或其他复杂交互。
- **计算开销未深度分析**：虽给出帧数对比，但未详细比较端到端时延，也未讨论树惰性构建外的加速优化。
- **泛化性局限**：尽管有迁移实验，但在其他风格的长视频（如监控、第一人称纪录）上仍有待验证，且 LongClueQA 数据集的质量和偏差未详细讨论。
