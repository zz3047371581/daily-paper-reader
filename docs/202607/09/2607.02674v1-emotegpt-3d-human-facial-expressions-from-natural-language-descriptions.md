## 摘要
从文本精确控制三维面部表情对虚拟化身、动画和人机交互至关重要，但现有的文本到三维方法同时生成身份、表情和纹理，导致难以实现细粒度表情控制。我们将文本驱动的表情合成构建为三维可变形模型（3DMM）解耦参数空间中的回归问题。然而，该设定需要将详细语言与精确表情参数关联的配对数据，而这在现有资源中尚缺。为填补这一空白，我们引入了Txt2Emote基准，包含多样化的三维面部表情，并配有通过GPT-4o和高保真面部跟踪器获取的细粒度文本注释，提供了描述面部特征的显式描述和涉及表情背后情境上下文的隐式描述。基于此数据集，我们提出了EmoteGPT，一个基于多模态大语言模型（MLLM）的文本到三维表情框架，通过专门的<Expr>标记对表情表示进行语义锚定，再将其解码为3DMM参数。我们进一步通过大规模图像到3DMM数据的增强训练来改进EmoteGPT，使其在情绪识别指标和感知表现力上超越了最先进的文本到三维面部合成方法。集成到化身流程中，我们的方法能够生成逼真和风格化的三维化身，以及从文本输入合成富有表现力且三维一致的二维面部图像。

## Abstract
Precise control of 3D facial expressions from text is crucial for virtual avatars, animation, and human-computer interaction, yet existing text-to-3D methods jointly generate identity, expression, and texture, making fine-grained expression control difficult. We instead formulate text-driven expression synthesis as a regression problem in the disentangled parameter space of a 3D Morphable Model (3DMM). This setting, however, requires paired data linking detailed language to precise expression parameters, which are missing from existing resources. To fill this gap, we introduce Txt2Emote, a benchmark of diverse 3D facial expressions with fine-grained textual annotations obtained from GPT-4o and a high-fidelity face tracker, providing both explicit descriptions detailing facial features and implicit descriptions referencing the situational context behind the expression. Leveraging this dataset, we present EmoteGPT, a text-to-3D expression framework based on a Multimodal Large Language Model (MLLM) with a dedicated <Expr> token to semantically ground expression representations, which are then decoded into 3DMM parameters. We further improve EmoteGPT by augmenting training with large-scale image-to-3DMM data, enabling it to surpass state-of-the-art text-to-3D face synthesis methods on emotion recognition metrics and in perceived expressiveness. Integrated into avatar pipelines, our method enables photorealistic and stylized 3D avatars, as well as expressive 3D-consistent 2D face synthesis from textual input.

    2.  **模型能力**：需要一个强大的模型来理解和关联复杂的语言语义与细粒度的面部肌肉运动。
*   **整体含义**：本文提出了一种新的范式，将**文本驱动的表情合成**构建为三维可变形模型（3DMM）参数空间中的一个**回归问题**，从而实现了更直观、更灵活且更高精度的三维表情控制。

## 2. 论文提出的方法论

论文的核心方法分为两部分：一个新基准数据集 `Txt2Emote` 和一个基于多模态大语言模型（MLLM）的框架 `EmoteGPT`。

*   **Txt2Emote 数据集**
    *   **构建基础**：基于 AffectNet 数据集，通过 EMOCAv2 重建三维人脸网格，并使用 GPT-4o 为每个表情生成两种类型的文本描述：
        *   **显式描述**：详细描述面部物理特征（如嘴型、眼部、眉毛）和整体情绪印象。
        *   **隐式描述**：描述能引发该表情的事件或情境。
    *   **数据规模**：包含 30,000 个表情样本，其中 2,500 个样本作为评估基准。

*   **EmoteGPT 框架**
    *   **核心思想**：将面部表情视为一种独特的“模态”，集成到 MLLM 的语言空间中。
    *   **模型架构**：
        1.  **骨干网络**：基于预训练的 MLLM（LLaVA-1.5-7B）。
        2.  **专用令牌 `<Expr>`**：扩展 MLLM 的词汇表，新增一个专门用于表示面部表情的 `<Expr>` 令牌。当输入文本或图像指令时，模型会输出该令牌，其对应的隐藏状态（ $h_{\langle Expr \rangle}$ ）即为表情的语义表示。
        3.  **表情解码器**：一个轻量级的 MLP 网络（ $\eta$ ），将 $h_{\langle Expr \rangle}$ 解码为 FLAME 模型的 3DMM 表情参数（ $\hat{\psi}$ ）。
    *   **训练策略**：
        1.  **多模态监督**：
            *   **文本-表情数据**：使用 Txt2Emote 数据集构造指令模板，直接监督映射学习。
            *   **图像-表情数据**：使用 CelebA、FFHQ 等大规模人脸数据集，通过 EMOCAv2 提取表情参数作为标签，构造“`<IMAGE>` 这张脸的表情参数是？”的指令模板，从而将图像信息作为额外的监督信号。
            *   **通用VQA数据**：引入 LLaVA-v1.5-mix665k 的指令跟随数据，以保持模型的通用推理能力。
        2.  **损失函数**：组合了三个损失项。
            *   **文本交叉熵损失**： $\mathcal{L} = \lambda_{txt} \text{CE}(y_{txt}, \hat{y}_{txt})$ ，用于保持模型的文本理解与生成能力。
            *   **参数空间 L2 损失**： $\lambda_{expr} |\psi - \hat{\psi}|^2_2$，直接监督表情参数的回归。
            *   **几何空间加权 L1 损失**： $\lambda_{mesh} |w \odot (M(\psi) - M(\hat{\psi}))|_1$，在三维网格顶点上计算损失，并对表情关键区域（如口、眼）进行加权，以约束几何形状。
        3.  **参数高效微调**：仅对 MLLM 使用 LoRA 微调，并训练表情解码器，其余参数冻结。

## 7. 优点

*   **问题定义新颖**：将文本到三维表情合成分解为 3DMM 参数空间中的回归问题，简化了任务，实现了表情与身份的完全解耦控制。

## 8. 不足与局限

*   **模型表征能力受限**：底层依赖 FLAME 模型，其训练数据覆盖的人种、年龄和微表情（如皱纹）有限，且无法做到完美的身份-表情解耦。

*   **对模糊输入的歧义性**：当输入的文本提示情绪不明确时，模型可能生成一个合理但强度较低的通用表情，而非多模态的复杂反应。
*   **表情维度单一**：该模型专注于静态表情合成，未考虑表情的时间动态、过渡以及与语音、头部姿态的协调。
